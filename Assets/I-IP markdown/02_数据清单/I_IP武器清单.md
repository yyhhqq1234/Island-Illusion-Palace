**四、武器系统整合说明**

**1. 武器类型分布**

-   **剑类**：3把（破晓之刃、熔火重剑、绯红记忆之刃）

-   **法杖**：4把（结晶法杖、灵魂引导者、冰霜法杖、灵魂共鸣法杖）

-   **镰刀**：4把（沼泽镰刀、雷霆镰刃、腐化吞噬者、绯红契约）

-   **晶能武装**：4把（晶能短刃、圣光裁决、时空裂隙匕首、岩地重锤）

**2. 元素属性分布**

-   **物理**：3把（破晓之刃、沼泽镰刀、岩地重锤）

-   **火焰**：2把（熔火重剑、绯红记忆之刃）

-   **冰霜**：2把（结晶法杖、冰霜法杖）

-   **雷电**：2把（晶能短刃、雷霆镰刃）

-   **灵魂**：3把（灵魂引导者、腐化吞噬者、灵魂共鸣法杖）

-   **圣光**：2把（圣光裁决、绯红契约）

-   **流水**：1把（时空裂隙匕首）

**3. 获取途径分布**

-   **敌人掉落**：9把（覆盖所有精英/BOSS敌人）

-   **地图宝箱/探索**：5把（覆盖主要地图类型）

-   **祭坛供奉**：1把（圣光裁决）

-   **剧情解锁**：1把（绯红契约）

**4. 与核心系统联动**

-   **灵魂负担系统**：灵魂共鸣法杖减缓负担增长

-   **召唤系统**：灵魂共鸣法杖和绯红契约强化召唤物

-   **弱点系统**：圣光裁决针对灵魂残留，腐化吞噬者针对腐化类敌人

-   **记忆碎片系统**：绯红记忆之刃与碎片收集联动

**五、重复武器获取处理规则**

当玩家已持有某武器，再次通过掉落/宝箱/祭坛等方式获取相同武器时，适用以下自动处理规则：

-   **自动分解**：重复获取的武器不会被加入背包，而是自动分解为灵魂资源
-   **灵魂返还 = 武器价值**：

  -----------------------------------------------------------------
  武器稀有度     灵魂返还量         说明
  -------------- ------------------ --------------------------------
  **普通（白）** 150                基础资源回收

  **精良（蓝）** 400                中级资源回收

  **史诗（紫）** 800                高级资源回收

  **传奇（金）** 1,500              顶级资源回收（绯红契约为唯一传奇武器）
  -----------------------------------------------------------------

-   **额外奖励：1次免费特效重刷机会**：
    -   重复获取武器时，除灵魂返还外，额外获得 **1次"特效重刷机会"**
    -   特效重刷机会：允许玩家在营火处对该武器已解锁的特效槽进行一次免费重刷（重刷无灵魂消耗，随机替换为同类型特效池中的另一特效）
    -   特效重刷机会**不叠加、不可跨武器使用**：仅适用于本次重复获取的武器类型（如重复获取"沼泽镰刀"则该机会仅可用于镰刀类特效重刷）；若玩家已拥有该武器的全部特效，则改为额外返还50%灵魂
    -   特效重刷机会在当次游戏中有效，离开游戏后未使用则失效
-   **设计意图**：避免重复武器成为无效掉落，同时赋予其策略价值——玩家可以选择消耗免费机会优化武器特效Build
