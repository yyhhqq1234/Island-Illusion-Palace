**二点五、灵魂获取倍率叠加规则（防双重惩罚）**

为避免高负担状态与高难度模式叠加后灵魂获取倍率过低（接近零收益感），明确以下规则：

**2.5.1 叠加方式：乘法叠加**

```
最终灵魂获取倍率 = 难度灵魂倍率 × 负担灵魂惩罚倍率
```

即各来源倍率独立计算后相乘，而非简单相加。此方式更符合"越叠加越惩罚"的直觉，且数值行为更可控。

**2.5.2 最低保底：不低于基础值的 35%**

```
最终灵魂获取倍率 = max(0.35, 难度灵魂倍率 × 负担灵魂惩罚倍率)
```

无论叠加多少惩罚倍率，最终灵魂获取量不低于基础值的 **35%**，确保玩家始终有可感知的收益。

**2.5.3 典型场景倍率计算**

| 场景 | 难度倍率 | 负担倍率 | 乘法结果 | 保底后 | 说明 |
|------|---------|---------|---------|-------|------|
| 简单 + 低负担 | 1.20 | 1.00 | 1.20 | 1.20 | 新手友好 |
| 普通 + 低负担 | 1.00 | 1.00 | 1.00 | 1.00 | 基准 |
| 困难 + 高负担(>70) | 0.80 | 0.70 | 0.56 | 0.56 | 挑战者惩罚 |
| 疯狂 + 低负担 | 0.70 | 1.00 | 0.70 | 0.70 | 纯难度惩罚 |
| **疯狂 + 高负担(>70)** | **0.70** | **0.70** | **0.49** | **0.49** | **最极端场景，仍高于35%保底** |
| 疯狂 + 临界负担(>90) | 0.70 | 0.70 | 0.49 | 0.49 | 同上 |
| *未来扩展*（如有额外惩罚） | *0.70* | *0.70* | *<0.35* | *0.35* | *保底生效* |

**2.5.4 设计意图**

- 保底 35% 确保极端情况下玩家仍有可感知的收益流，避免"零收益"的心理挫败
- 当前最极端场景（疯狂+高负担=0.49）未触发保底，保底作为安全网防止未来扩展时出现极端惩罚
- 负担系统已通过多重方式惩罚高负担玩家（属性惩罚、敌人侵略性、时空裂隙），资源获取惩罚不应叠加到失去游戏意义

**7.1 难度管理器（新建 DifficultyManager.cs）**

```pseudocode
// 文件：新建 Assets/Scripts/Systems/DifficultyManager.cs
// 实现路径：全新创建，挂载在Persistent GameObject上（DontDestroyOnLoad）

public enum DifficultyMode { Easy, Normal, Hard, Hardcore }

public class DifficultyManager : MonoBehaviour:
    // ★ 单例模式（跨场景持久化）
    public static DifficultyManager Instance
    
    public DifficultyMode currentMode = Normal
    public DifficultyConfig config  // ScriptableObject配置
    
    void Awake():
        IF Instance == null:
            Instance = this
            DontDestroyOnLoad(gameObject)
        ELSE:
            Destroy(gameObject)
    
    // ★ 获取当前难度倍率
    FUNCTION GetEnemyHealthMultiplier() -> float:
        SWITCH currentMode:
            CASE Easy:     RETURN 0.80
            CASE Normal:   RETURN 1.00
            CASE Hard:     RETURN 1.30
            CASE Hardcore: RETURN 1.50
    
    FUNCTION GetEnemyDamageMultiplier() -> float:
        SWITCH currentMode:
            CASE Easy:     RETURN 0.80
            CASE Normal:   RETURN 1.00
            CASE Hard:     RETURN 1.30
            CASE Hardcore: RETURN 1.50
    
    FUNCTION GetSoulDropMultiplier() -> float:
        SWITCH currentMode:
            CASE Easy:     RETURN 1.20
            CASE Normal:   RETURN 1.00
            CASE Hard:     RETURN 0.80
            CASE Hardcore: RETURN 0.70
    
    // ★ 灵魂获取最终倍率（含负担惩罚叠加，乘法+保底）
    FUNCTION GetEffectiveSoulMultiplier(burdenMultiplier: float) -> float:
        rawMultiplier = GetSoulDropMultiplier() * burdenMultiplier
        RETURN Mathf.Max(0.35, rawMultiplier)  // 保底不低于基础值的35%
    
    FUNCTION GetBurdenGainMultiplier() -> float:
        SWITCH currentMode:
            CASE Easy:     RETURN 0.70
            CASE Normal:   RETURN 1.00
            CASE Hard:     RETURN 1.50
            CASE Hardcore: RETURN 2.00
    
    // ★ 特殊机制开关
    FUNCTION IsEnemyBerserkEnabled() -> bool:
        RETURN currentMode == Hardcore
    
    FUNCTION IsTimeCollapseEnabled() -> bool:
        RETURN currentMode == Hardcore  // 每2分钟随机负面效果
    
    FUNCTION IsSoulCorrosionEnabled() -> bool:
        RETURN currentMode == Hardcore  // 30%灵魂变负担
    
    // ★ 难度解锁检查
    FUNCTION CanSelectMode(mode: DifficultyMode) -> bool:
        SWITCH mode:
            CASE Easy:     RETURN true          // 初始解锁
            CASE Normal:   RETURN true          // 初始解锁
            CASE Hard:     RETURN hasCompletedNormal  // 通关普通
            CASE Hardcore: RETURN hasCompletedHard    // 通关困难
```

**7.2 难度在敌人生成时的应用（EnemySpawner + DifficultyManager）**

```pseudocode
// 文件：Assets/Scripts/Enemy/EnemySpawner.cs
// 实现路径：在SpawnEnemy中乘以难度倍率

FUNCTION SpawnEnemy(enemyPrefab: GameObject, position: Vector3) -> GameObject:
    enemy = Instantiate(enemyPrefab, position)
    
    // ★ 应用难度倍率
    diff = DifficultyManager.Instance
    IF diff != null:
        enemyAI = enemy.GetComponent<EnemyAI>()
        enemyAI.damage *= diff.GetEnemyDamageMultiplier()
        enemyAI.maxHealth *= diff.GetEnemyHealthMultiplier()
        
        // 困难模式：弱点不可见
        IF diff.currentMode >= Hard:
            enemyAI.hideWeaknesses = true
        
        // 疯狂模式：所有敌人50%血自动狂暴
        IF diff.IsEnemyBerserkEnabled():
            enemyAI.autoBerserkAtHalfHP = true
    
    RETURN enemy
```

```pseudocode
// 文件：新建 Assets/Scripts/UI/DifficultySelectUI.cs
// 实现路径：主菜单→"新游戏"→弹出难度选择面板

FUNCTION ShowDifficultyPanel():
    // 显示4个难度按钮
    FOR EACH mode IN [Easy, Normal, Hard, Hardcore]:
        button = CreateDifficultyButton(mode)
        
        IF DifficultyManager.CanSelectMode(mode):
            button.interactable = true
            button.text = GetDifficultyName(mode)
        ELSE:
            button.interactable = false
            button.text = GetDifficultyName(mode) + "（锁定）"
            button.hint = GetUnlockCondition(mode)
    
    // 默认高亮Normal
    HighlightButton(Normal)

FUNCTION OnDifficultySelected(mode: DifficultyMode):
    DifficultyManager.Instance.currentMode = mode
    // 加载GamePlay场景开始游戏
    SceneManager.LoadScene("GamePlay")
```
