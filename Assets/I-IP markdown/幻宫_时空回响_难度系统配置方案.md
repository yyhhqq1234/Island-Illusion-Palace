**二、按难度档位精确参数**

**2.1 简单模式**

  ----------------------------------------------------------------------------------------------------------
  参数类别                 具体调整                                                           设计意图
  ------------------------ ------------------------------------------------------------------ ------------------------------
  **敌人属性（全局）**     伤害倍率 0.80×、生命倍率 0.80×、移动速度倍率 0.90×                容错率高，适合学习操作

  **敌人 AI**              攻击间隔 +25%（更慢），追击范围 -20%，伏击触发距离 -30%             给玩家更多反应时间

  **资源获取**             灵魂掉落倍率 1.20×、材料掉落倍率 1.10×、精华掉落倍率 0.80×          加速成长，降低挫败

  **负担系统**             负担增长速率 ×0.70，营火清除负担量不变，负担≥70时无属性惩罚         降低策略压力

  **BOSS 战**              BOSS 阶段数 -1（如原3阶段简化为2阶段），攻击前摇 +30%               减少认知负担

  **死亡惩罚**             死亡后仅损失当前灵魂的 10%（最低 50），无额外惩罚                    轻度惩罚

  **新手保护**             前 5 次炼金失败返还 90%-100% 材料，前 3 次死亡免除惩罚              极低挫败

  **永久死亡**             不启用（玩家不会因死亡失去进度）                                    -
  ----------------------------------------------------------------------------------------------------------

**2.2 普通模式**

  ----------------------------------------------------------------------------------------------------------
  参数类别                 具体调整                                                           设计意图
  ------------------------ ------------------------------------------------------------------ ------------------------------
  **敌人属性（全局）**     所有倍率为标准 1.0×（与敌人数据清单一致）                          基准体验

  **负担系统**             ≥70 高负担（属性-20%），≥90 临界负担（属性-40%），100 强制传送     引入核心风险决策

  **BOSS 战**              使用 BOSS 数据清单中列出的完整阶段数                               标准 BOSS 体验

  **死亡惩罚**             损失当前灵魂的 20%（最低 100），在主菜单复活                        标准惩罚

  **新手保护**             前 3 次炼金失败返还 80%-90% 材料                                   适度保护

  **永久死亡**             不启用                                                             -
  ----------------------------------------------------------------------------------------------------------

**2.3 困难模式**

  ----------------------------------------------------------------------------------------------------------
  参数类别                 具体调整                                                           设计意图
  ------------------------ ------------------------------------------------------------------ ------------------------------
  **敌人属性（全局）**     伤害×1.30、生命×1.30、移动速度×1.20                                显著提升挑战

  **负担系统**             负担增长速率 ×1.50，恢复速率 ×0.70，≥50 即触发轻微属性惩罚         负担管理更严格

  **BOSS 战**              BOSS 增加 1 个额外攻击模式，阶段转换时获得短暂增益                  更多变化

  **死亡惩罚**             损失当前灵魂的 30%（最低 200），复活后在营火重生                    更高惩罚

  **新手保护**             无新手保护                                                         纯粹挑战

**三、动态难度调整**

**3.1 基于玩家表现的浮动**

  ----------------------------------------------------------------------------------------------------------
  触发条件                      调整内容                              调整幅度          持续时间
  ----------------------------- ------------------------------------- ----------------- ------------------
  **连胜 ≥ 5 场**               下一区域敌人属性 +10%                  全局 ×1.10        至下一场失败

  **连胜 ≥ 10 场**              下一区域敌人属性 +20%，精英概率 +30%   全局 ×1.20        至下一场失败

  **连败 ≥ 3 场**               下一区域敌人属性 -10%                  全局 ×0.90        至下一场胜利

  **快速清图（< 3 分钟/区域）**  下一区域精英密度 +40%                 仅下一区域        仅限下一区域

  **BOSS 无伤击败**             掉落奖励 +50%                         仅该次掉落        仅该次

  **死亡后复活**                临时全属性 +10%（仅持续 3 场战斗）     局部              3 场战斗
  ----------------------------------------------------------------------------------------------------------

**3.2 基于负担值的动态**

  -------------------------------------------------------------------------
  负担值范围           动态效果
  -------------------- ----------------------------------------------------
  **< 30**             敌人侦查范围 -10%，偷袭概率降低

  **30 - 70**          标准行为

  **70 - 90**          时空裂隙出现概率 +10%，敌人侵略性 +15%

  **> 90**             敌人视为玩家"已标记"，全图敌人向玩家靠拢
  -------------------------------------------------------------------------

**3.3 基于游戏进度的动态**

  ---------------------------------------------------------------------------
  游戏进度（区域数）   调整内容                             设计意图
  -------------------- ------------------------------------ -------------------
  **前 2 区域**        敌人配置简单，弱点明显可见           教学阶段

  **3-5 区域**         引入组合敌人，隐藏弱点              提升挑战

  **6+ 区域**          精英频繁出现，环境威胁增加           终局准备

  **记忆区域**         敌人强度 = 当前区域 + 5 级           关键挑战

  **最终区域**         固定配置（不受动态调整影响）         公平决战
  ---------------------------------------------------------------------------

**四、难度选择 UI 流程**

1.  **新游戏**：弹出难度选择界面，默认高亮"普通"，简单和普通直接可选，困难和疯狂灰色（显示解锁条件）。

2.  **继续游戏**：自动加载上次使用的难度。

3.  **多周目**：开局时重新选择难度，已解锁的难度全部可用。

4.  **难度切换**：单局内不可切换难度，需重新开始新周目。

**五、多周目难度扩展**

  ---------------------------------------------------------------------------------
  周目数      难度解锁               推荐难度            敌人额外变化
  ----------- ---------------------- ------------------- ---------------------------
  **1 周目**  简单、普通              普通                无额外变化

  **2 周目**  简单、普通、困难        困难                敌人组合变化，弱点隐蔽

  **3 周目**  简单、普通、困难、疯狂  困难/疯狂           精英频繁，特殊机制激活

  **4 周目+** 全部                   疯狂                完全随机敌人配置
  ---------------------------------------------------------------------------------

**六、难度参数可供开发者调整的 Inspector 暴露项（代码侧）**

```csharp
[System.Serializable]
public class DifficultyConfig
{
    [Header("敌人属性倍率")]
    [Range(0.5f, 2f)] public float enemyDamageMultiplier = 1f;
    [Range(0.5f, 2f)] public float enemyHealthMultiplier = 1f;
    [Range(0.5f, 2f)] public float enemySpeedMultiplier = 1f;
    
    [Header("资源掉落倍率")]
    [Range(0.1f, 3f)] public float soulDropMultiplier = 1f;
    [Range(0.1f, 3f)] public float essenceDropMultiplier = 1f;
    [Range(0.1f, 3f)] public float materialDropMultiplier = 1f;
    
    [Header("负担倍率")]
    [Range(0.1f, 3f)] public float burdenGainMultiplier = 1f;
    [Range(0.1f, 3f)] public float burdenRecoveryMultiplier = 1f;
    
    [Header("特殊机制开关")]
    public bool enableEnemyBerserk = false;
    public bool enableTimeCollapse = false;
    public bool enableSoulCorrosion = false;
    
    [Header("门槛")]
    public float burdenPenaltyThreshold = 70f;
    public float deathSoulLossRate = 0.2f;
    public int deathSoulLossMinimum = 100;
}
```
