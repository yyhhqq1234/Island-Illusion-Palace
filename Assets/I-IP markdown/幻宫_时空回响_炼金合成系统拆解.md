**《幻宫：时空回响》炼金合成系统详细拆解（简化探索版）**

**一、 系统核心定位与原则**

-   **定位**：自由探索、合成驱动的策略支持系统，为战斗、探索与资源管理提供核心支持。

-   **核心原则**：

    1.  **配方探索**：游戏仅提供4个基础配方（治疗药剂、灵魂稳定剂、召唤持续时间药剂、负担缓解熏香），其余10个配方需玩家通过组合材料自主发现。

    2.  **材料可合成**：玩家可通过组合低价值材料，合成高价值材料（包括稀有、史诗材料）。

    3.  **记忆碎片催化**：携带任意记忆碎片可提升炼金成功率，无碎片时无惩罚。

    4.  **高风险高回报**：合成高价值材料成功率较低，失败时按**素数分解机制**返还部分材料。

    5.  **系统简化**：无炼金等级、材料亲和等成长线，专注于探索与合成。

**二、 材料与配方体系**

**2.1 材料分类（沿用原清单）**

  ------------------------------------------------------------------------------------------------------
  类别           价值范围   数量   是否可合成           说明
  -------------- ---------- ------ -------------------- ------------------------------------------------
  **基础材料**   1--10      10     否（仅采集/掉落）    如腐朽木屑、星光草、灵魂微尘等

  **稀有材料**   11--20     10     是                   如记忆残渣、时空碎片、灵魂结晶等

  **史诗材料**   30, 33, 1  3      部分（除灵魂精华）   地脉结晶、远古铭文石可合成；灵魂精华为炼金产物

  **传奇材料**   40         1      否（仅掉落）         悖时薄片
  ------------------------------------------------------------------------------------------------------

**2.2 配方分类**

-   **基础配方（4个）**：游戏开始时已知。

-   **隐藏配方（10个）**：需玩家探索发现的物品配方，包括7个进阶配方（如召唤强化剂、弱点洞察药剂等）和3个传奇配方（如时空道标、负担净化圣水等）。所有隐藏配方均为物品配方，共计10个。

**三、 合成流程与逻辑**

**3.1 合成界面与操作**

1.  玩家在炼金台放入**2-4种材料**。

2.  系统按以下优先级判断合成目标：

    -   **优先**：匹配物品配方（包括已知与隐藏）。

    -   **次之**：若总价值匹配可合成材料（价值为11-20、30、33），则尝试通过材料合成机制合成该材料。（注：材料合成为独立机制，不计入上述14个物品配方中）

    -   **失败**：若均不匹配，合成失败，按**总价值素数分解**返还基础材料。

**3.2 合成成功率公式**

-   **物品合成**：

text

基础成功率 = 70% + (智慧 × 1%) - (当前负担 × 0.2%)

（若携带任意记忆碎片，最终成功率 +15%）

-   **材料合成**：

text

基础成功率 = 80% - (目标材料价值 - 10) × 2%

（若携带任意记忆碎片，最终成功率 +15%）

-   **成功率上下限**：5% -- 95%

**3.3 合成结果**

  ----------------------------------------------------------------------------------------------------------------------------
  合成类型       成功                                                           失败
  -------------- -------------------------------------------------------------- ----------------------------------------------
  **物品合成**   获得对应物品；若成功率≥95%触发**大成功**（效果+20%或数量+1）   材料全损，按**材料返还机制**返还50%~70%价值的基础材料，额外获得"**炼金残渣**"

  **材料合成**   获得目标材料                                                   材料全损，按**材料返还机制**返还50%~70%价值的基础材料
  ----------------------------------------------------------------------------------------------------------------------------

**四、 材料返还机制**

**4.1 核心规则**

合成失败时，系统根据投入材料的**总价值**，按**价值匹配原则**返还**基础材料**（价值1-10），返还总价值为投入总价值的 **50%~70%**（随机）。

**4.2 返还逻辑**

1.  **计算投入总价值**：将放入的所有材料价值相加。

2.  **计算返还价值**：返还总价值 = 投入总价值 × (50%~70%随机)。

3.  **基础材料组合**：使用基础材料（价值1-10）组合成返还总价值，优先选择高价值基础材料。

    -   可用基础材料：腐朽木屑(1)、碎骨(2)、星光草(3)、腐化组织(4)、月影花(5)、锈蚀零件(6)、灵魂微尘(7)、浑浊露珠(8)、净化盐晶(9)、哀嚎藤蔓(10)

4.  **材料数量限制**：每个基础材料最多返还1个，确保返还多样性。

**4.3 返还算法**

```text
输入：投入总价值 V，返还比例 R (0.5~0.7)
输出：基础材料列表

1. 目标返还价值 = V × R
2. 从最高价值基础材料（哀嚎藤蔓10）开始向下遍历
3. 若当前材料价值 ≤ 剩余返还价值，加入返还列表，扣除对应价值
4. 重复步骤3，直到剩余返还价值 < 1或遍历完所有基础材料
5. 若仍有剩余价值（<1），随机赠送1个最低价值材料（腐朽木屑）
```

**4.4 示例**

-   合成"治疗药剂"（投入材料总价值8）失败：
    -   返还比例：假设60%
    -   返还价值：8 × 60% = 4.8
    -   返还材料：腐化组织(4) ×1（剩余0.8，不足1，赠送腐朽木屑(1)×1）
    -   最终返还：腐化组织×1、腐朽木屑×1

-   合成"灵魂稳定剂"（投入材料总价值25）失败：
    -   返还比例：假设55%
    -   返还价值：25 × 55% = 13.75
    -   返还材料：哀嚎藤蔓(10)×1、灵魂微尘(7)×1（10+7=17 > 13.75，调整为哀嚎藤蔓(10)×1 + 星光草(3)×1 = 13）
    -   最终返还：哀嚎藤蔓×1、星光草×1

-   合成"时空道标"（投入材料总价值76）失败：
    -   返还比例：假设65%
    -   返还价值：76 × 65% = 49.4
    -   返还材料：哀嚎藤蔓(10)×1、净化盐晶(9)×1、浑浊露珠(8)×1、灵魂微尘(7)×1、锈蚀零件(6)×1、月影花(5)×1、腐化组织(4)×1（10+9+8+7+6+5+4=49）
    -   最终返还：哀嚎藤蔓×1、净化盐晶×1、浑浊露珠×1、灵魂微尘×1、锈蚀零件×1、月影花×1、腐化组织×1

**4.5 特殊规则**

-   **新手保护**：玩家前3次合成失败时，返还比例提升至 **80%~90%**。
-   **材料偏好**：返还时优先选择玩家当前数量最少的基础材料类型。
-   **炼金残渣**：物品合成失败时额外获得1个"炼金残渣"（可用于基础配方）。

**五、 配方探索与发现**

**5.1 物品配方探索**

-   玩家通过尝试不同材料组合，发现隐藏的10个物品配方。

-   系统在首次合成成功时，永久记录该配方至**炼金笔记**。

**5.2 材料合成配方探索**

-   每个可合成材料（价值11-20、30、33）有**1个固定合成配方**（由多种材料组成，总价值等于目标价值）。

-   玩家通过尝试不同材料组合，当总价值匹配目标材料价值时，即有可能发现该配方（仍需成功率判定）。

**5.3 探索线索**

-   **材料描述**：隐含属性与用途线索。

-   **环境叙事**：实验室碎片区域的书架、笔记等提供暗示。

-   **炼金笔记**：记录已发现配方，支持玩家回溯与推理。

**六、 与其他系统的协同**

**6.1 与负担系统**

-   高负担降低炼金成功率。

-   炼金产物（如灵魂稳定剂、负担净化圣水）是管理负担的核心手段。

**6.2 与战斗系统**

-   提供战斗前（弱点洞察）、中（治疗）、后（负担缓解）支持。

-   部分配方（如机械仆从核心）直接提供战斗单位。

**6.3 与资源经济**

-   材料作为第四类资源，形成"采集/掉落 → 合成 → 使用/再合成"循环。

-   炼金残渣可作为低价值材料用于部分基础配方。

**6.4 与探索系统**

-   不同区域产出特定材料，驱动探索。

-   传奇配方产物（如时空道标）极大提升探索效率。

```pseudocode
FUNCTION CraftWithMaterials(materials: MaterialTypeEnum[]) -> bool:
    pendingProduce.Clear()  // 清空待收集产出
    
    // 1. 材料校验
    FOR EACH mat IN materials:
        IF NOT materialInventory.ContainsKey(mat) OR materialInventory[mat] <= 0:
            PlayAudio("PlayAlchemyFail")
            RETURN false
    
    // 2. 计算成功率
    successRate = CalculateFreeAlchemySuccessRate(materials)
    
    // 3. 消耗材料
    FOR EACH mat IN materials:
        RemoveMaterial(mat, 1)
    
    // 4. 判定成功/失败
    IF Random.value <= successRate:
        GenerateRandomResult(materials)      // 成功：生成产物
        PlayAudio("PlayAlchemySuccess")
        RETURN true
    ELSE:
        failedAttempts++
        RefundMaterialsForFreeAlchemy(materials)  // 失败：素数值分解返还
        pendingProduce.Add(MemoryResidue)   // 失败赠送记忆残渣
        PlayAudio("PlayAlchemyFail")
        RETURN false
```

```pseudocode
// 文件：Assets/Scripts/Systems/AlchemySystem.cs

FUNCTION MatchRecipeByMaterials(materials: MaterialTypeEnum[]) -> RecipeType:
    // 对材料排序并构建频率字典
    materialCounts = Dictionary<MaterialTypeEnum, int>()
    FOR EACH m IN materials:
        materialCounts[m] = materialCounts.GetOrDefault(m, 0) + 1
    
    // 遍历所有配方（包括隐藏配方）
    FOR EACH recipe IN availableRecipes:
        IF recipe.requiredMaterials.Count != materialCounts.Count: CONTINUE
        
        isMatch = true
        FOR EACH (mat, count) IN recipe.requiredMaterials:
            IF materialCounts.GetOrDefault(mat, 0) != count:
                isMatch = false; BREAK
        
        IF isMatch AND recipe.type != matchedRecipe:  // 确保唯一匹配
            matchedRecipe = recipe.type
    
    RETURN matchedRecipe  // 无匹配返回 RecipeType.None
```

**9.3 成功率计算（AlchemySystem.CalculateSuccessRate）**

```pseudocode
// 文件：Assets/Scripts/Systems/AlchemySystem.cs
// 实现路径：已有实现，需确认Inspector可配置参数已暴露

FUNCTION CalculateSuccessRate(recipe: RecipeData) -> float:
    baseRate = recipeBaseSuccessRate  // 默认0.70（Inspector可配）
    
    // 智慧加成（每点+1%）
    IF characterStats != null:
        baseRate += characterStats.intelligence * 0.01
    
    // 负担惩罚（每点-0.2%）
    IF burdenSystem != null:
        baseRate -= burdenSystem.currentBurden * 0.002
    
    // 记忆碎片加成（携带任意碎片+15%）
    IF HasMemoryFragment():
        baseRate += memoryShardBonus  // 默认0.15
    
    // ★ 材料合成特殊公式：80% - (价值-10)×2%
    IF IsMaterialRecipe(recipe.type):
        baseRate = materialCraftSuccessRate  // 默认0.80
        baseRate -= (recipe.alchemicalValue - 10) * 0.02
    
    // 新手保护：前3次失败减免
    IF failedAttempts < 3:
        reduction = (3 - failedAttempts) / 3.0
        baseRate = Lerp(baseRate, 0.90, reduction)
    
    RETURN Clamp(baseRate, successRateMinimum, successRateMaximum)  // 5%~95%
```

**9.4 自由炼金成功率计算（AlchemySystem.CalculateFreeAlchemySuccessRate）**

```pseudocode
// 实现路径：需从材料稀有度/总价值计算成功率

FUNCTION CalculateFreeAlchemySuccessRate(materials: MaterialTypeEnum[]) -> float:
    // 1. 计算总价值
    totalValue = SUM(GetMaterialValue(m) FOR EACH m IN materials)
    
    // 2. 基础成功率：70% - 每超出10价值降低5%
    baseRate = freeAlchemyBaseSuccessRate - MAX(0, (totalValue - 20) * 0.005)
    
    // 3. 智慧加成
    IF characterStats != null:
        baseRate += characterStats.intelligence * 0.01
    
    // 4. 负担惩罚
    IF burdenSystem != null:
        baseRate -= burdenSystem.currentBurden * 0.002
    
    // 5. 记忆碎片加成
    IF HasMemoryFragment():
        baseRate += memoryShardBonus
    
    // 6. 新手保护
    IF failedAttempts < 3:
        baseRate = Lerp(baseRate, 0.90, (3 - failedAttempts) / 3.0)
    
    RETURN Clamp(baseRate, successRateMinimum, successRateMaximum)
```

**9.5 材料返还机制（AlchemySystem.RefundMaterials / RefundMaterialsForFreeAlchemy）**

```pseudocode
// 文件：Assets/Scripts/Systems/AlchemySystem.cs
// 实现路径：已有实现，此处为标准参考

FUNCTION RefundMaterials(recipe: RecipeData):
    // 1. 计算投入总价值
    totalValue = 0
    FOR EACH (mat, count) IN recipe.requiredMaterials:
        totalValue += GetMaterialValue(mat) * count
    
    // 2. 返还比例（新手保护：80-90%，正常：50-70%）
    IF failedAttempts < 3:
        refundRate = Random.Range(0.8, 0.9)
    ELSE:
        refundRate = Random.Range(0.5, 0.7)
    
    // 3. 返还价值 = 总价值 × 返还比例
    refundValue = Max(1, Floor(totalValue * refundRate))
    
    // 4. 用基础材料（价值1-10）从高到低填充返还价值
    basicMaterials = [哀嚎藤蔓(10), 净化盐晶(9), 浑浊露珠(8), 
                      灵魂微尘(7), 锈蚀零件(6), 月影花(5), 腐化组织(4),
                      星光草(3), 碎骨(2), 腐朽木屑(1)]
    
    remaining = refundValue
    refundList = []
    FOR EACH mat IN basicMaterials:  // 从高到低
        value = GetMaterialValue(mat)
        IF value <= remaining:
            refundList.Add(mat)
            remaining -= value
    
    // 5. 剩余价值 < 1 时赠送腐朽木屑
    IF remaining > 0:
        refundList.Add(RottenWood)
    
    // 6. 添加到库存
    FOR EACH mat IN refundList:
        AddMaterial(mat, 1)
    
    // 7. ★ 物品合成失败额外获得"炼金残渣"（记忆残渣）
    IF NOT IsMaterialRecipe(recipe.type):
        AddMaterial(MemoryResidue, 1)
```

```pseudocode
    FOR EACH item IN produceItems:
        IF item IS MaterialTypeEnum: inventorySystem.AddMaterial(item, 1)
        ELSE IF item IS RecipeType: inventorySystem.AddPotion(item, 1)
    produceItems.Clear()
```
