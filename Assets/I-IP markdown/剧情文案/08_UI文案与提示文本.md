| 2 | 碎片世界在你眼前崩散。然后，营火的温度重新包裹你。有些路需要多走几次才能看清——这不可耻。 |

## 四、游戏内核心UI提示

> **使用规则**：首次触发对应功能时显示，使用第二人称"你"。

### 4.1 首次进入游戏

| 提示ID | 触发条件 | 提示文本 |
|--------|---------|---------|
| UI-FIRST-01 | 首次进入碎片世界 | 你在一片世界碎片中醒来。戒指在牵引你——跟着它的微光走。前方有一簇营火。 |
| UI-FIRST-02 | 首次到达营火 | 营火：在此休息可恢复生命与负担。墨语会在火光中陷入沉思——那些收集的记忆碎片，会在这里找到表达。 |
| UI-FIRST-03 | 首次收集记忆碎片 | 莎娜的记忆碎片——每当收集一片，墨语会看到她的过去。在营火处可以重温这些记忆，或将它们激活以获得共鸣增益。 |

### 4.2 系统功能提示

| 提示ID | 触发条件 | 提示文本 |
|--------|---------|---------|
| UI-SYS-01 | 首次激活记忆碎片 | 激活记忆碎片将获得共鸣增益，但会增加负担。你最多可同时激活3个碎片。 |
| UI-SYS-02 | 负担首次超过70 | 负担已经很高了。戒指在反噬你，世界在排斥你。尽快找到营火——或使用负担缓解物品。 |
| UI-SYS-03 | 首次进入Boss房间 | 前方有强大的存在。击败它，你将获得核心碎片和稳定传送门——这是修复世界的关键步骤。 |
| UI-SYS-04 | 首次使用召唤 | 按下R键打开召唤轮盘选择灵魂之核。你最多可同时召唤3个生物。每个召唤物都有对应元素的弱点与抗性。 |
| UI-SYS-05 | 首次发现炼金材料 | 这种材料可用于炼金合成。在营火或炼金锅处，将材料组合以发现配方——炼金术是创之人教给墨语的技术。 |
| UI-SYS-06 | 首次进入安全区域 | 你已经进入安全区域。晶的侵蚀在这里被压制，世界的排斥也暂时减轻。在此可以放心探索。 |
| UI-SYS-07 | 首次发现时空裂隙 | 前方有一个不稳定的时空裂隙——它会在一段时间后消失。进入裂隙可以穿越到另一个世界碎片。 |
| UI-SYS-08 | 世界修复程度首次提升 | 世界修复程度提升了。随着修复程度的提高，世界对你的排斥会逐渐减轻，碎片之间的连接也会更加稳定。 |
