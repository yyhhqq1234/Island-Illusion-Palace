# 09 — 物品与图鉴文案

> **文件说明**：14个炼金配方背景故事、24种材料描述文案、3类敌人图鉴文本。
> **叙事基调**：破碎中的希望——物品文案兼具世界感与功能性。
> **语言风格**：遵循《幻宫语言风格指南》UI语体规范——炼金配方背景30-50字，材料描述15-25字，敌人图鉴30-50字。富有世界感但精炼克制。

---

## 一、14个炼金配方背景故事

> **设计说明**：每个配方附带一段叙事文本，暗示配方背后的故事或含有该配方信息的人。记录于墨语的炼金笔记中。

### 1.1 基础配方（4个）

| 配方名称 | 背景故事 |
|---------|---------|
| **灵魂稳定剂** | 创之人教给墨语的第一个炼金配方。他说："灵魂法术留下的伤口，得用世界上最小的盐来缝合。"那盐，就是净化盐晶。 |
| **治疗药剂** | 伊卡洛斯在墨语第一次被创之人的锤子砸到手时冲进药房，三分钟后端出来的东西。后来她承认配方是从一本皱巴巴的草药手册里抄的。 |
| **召唤持续时间药剂** | 绯红家族某个早已失名的先祖留下的配方。她在笔记里写："召唤物不是工具。它们的时间，是你的馈赠。" |
| **负担缓解熏香** | 月影花本是葬礼用的花。创之人发现它在锻炉旁燃烧时能让人心静。"有时候，最难锻造的不是铁——是自己的状态。" |

### 1.2 进阶配方（7个）

| 配方名称 | 背景故事 |
|---------|---------|
| **召唤强化剂** | 一个炼金术士在战场废墟里发现的配方——写在破损的铠甲内侧。铠甲主人用它强化了召唤物，但没能活到战后。 |
| **弱点洞察药剂** | 据说配方出自一位盲人炼金师。她看不见敌人的弱点——所以她造了能看到的东西。 |
| **记忆共鸣药剂** | 莎娜研究死灵魔法时偶然合成的配方。她在笔记边缘画了一个小箭头，写了两个字："有用。"——她从不写多余的字。 |
| **机械仆从核心** | 在世界撕裂前的南方国度边境，有一群工匠用废弃机械零件制造仆从来守护村庄。这个配方是他们的遗产——唯一的幸存者把它刻在石板上，埋进了废墟。 |
| **环境稳定卷轴** | 世界撕裂后，碎片中的环境变得不稳定——重力翻转，时间扭曲，空间折叠。一位在碎片中幸存的老学者，在营火旁写了这个配方。纸是烧焦的，墨是营火的灰。 |
| **同频强化药剂** | 墨语在创之人的旧笔记里找到的。笔记上画了一把剑和一个圆圈，中间写着："当锤子和材料共振时，锻造才真正开始。"他把这个原理用在了召唤物上。 |
| **重生护符** | 绯红家族的地下密室里有十二个这样的护符——每一个对应一位历代继承者。第十二个是空的，位置留给了莎娜。她没有用——但她留下了配方。 |

### 1.3 传奇配方（3个）

| 配方名称 | 背景故事 |
|---------|---------|
| **时空道标** | 悖时薄片——世界撕裂时产生的异常物质。时间和空间在薄片中被折叠，每一片都包含一个"不可能存在的地点"。将它们锚定，就能在碎片之间创建一个永久的锚点。 |
| **真理窥视药剂** | 远古铭文石上记录了"观测"的概念——不是用眼睛，是用某种超越感官的方式。墨语将这个古老知识与炼金术结合，创造了一个揭示隐藏事物的配方。铭文石的最后一句话他没读懂——那是编号099宇宙的文字。 |
| **负担净化圣水** | 地脉结晶——大地能量脉络中凝结的坚固结晶。在世界撕裂之前，它是地下的沉默力量。在世界撕裂之后，它是少数能对抗双重负担的物质。墨语花了三个碎片周期才找到正确的配比。 |

## 二、24种材料描述文案

> **设计说明**：每种材料的描述文案15-25字，精炼且富有世界感。

### 2.1 基础材料（10种）

| 材料名称 | 描述文案 |
|---------|---------|
| **腐朽木屑** | 枯朽的木材碎片。曾被火焰遗忘，如今在碎片中沉默。 |
| **碎骨** | 断裂的骨骼。不知是谁的，也不知断裂时疼不疼。 |
| **星光草** | 夜晚发光的植物。在世界碎裂后，它仍在努力发光。 |
| **腐化组织** | 被碎片能量侵蚀的肉块。残留着最后一丝不属于晶的温热。 |
| **月影花** | 满月夜绽放的银色花朵。花瓣上有微小的晶斑——它正在被同化，但还没有。 |
| **锈蚀零件** | 废弃机械的金属零件。在世界撕裂那天停止了运转。 |
| **灵魂微尘** | 逸散的灵魂能量微粒。风一吹就散，聚在一起就是故事。 |
| **浑浊露珠** | 凝结时空能量的水珠。每一滴都吞下了一小口时间。 |
| **净化盐晶** | 蕴含净化能量的盐结晶。创之人说过——最小的盐，缝最深的伤。 |
| **哀嚎藤蔓** | 触碰时会发出悲鸣的黑暗植物。不是它在哭——是被它吸收的痛苦在哭。 |

### 2.2 稀有材料（10种）

| 材料名称 | 描述文案 |
|---------|---------|
| **记忆残渣** | 记忆逸散的能量渣滓。细看时有画面闪动，都是别人的故事。 |
| **时空碎片** | 包含不稳定时空能量的脆弱碎片。手握时能感到时间在指尖打滑。 |
| **灵魂结晶** | 提纯凝固的灵魂能量块。半透明，内部有微弱的蓝色脉动。 |
| **晶化残核** | 晶化生物遗留的惰性能量核。晶已经死了，但形状还在。 |
| **机械核心** | 古代机械的微型动力核心。还在微弱地震动——像一颗不会停的心。 |
| **古树树脂** | 古老树木渗出的生命力树脂。在世界碎裂前就开始流淌了。 |
| **熔岩核心** | 一小块永不熄灭的炽热能量核。在世界撕裂时从地心喷涌而出。 |
| **石像鬼碎片** | 活化雕像上剥落的带魔力石片。碎片背面刻着无人能识的铭文。 |
| **极寒冰屑** | 永恒冻土中形成的刺骨冰屑。据说每一片都冻住了一个未说完的词。 |
| **古龙骨粉** | 研磨自远古巨兽骨骼的能量粉末。在大地深处沉睡了比契约更久的时间。 |

### 2.3 史诗材料（3种）

| 材料名称 | 描述文案 |
|---------|---------|
| **灵魂精华** | 五十个灵魂的提纯精华。不是你收集的——是他们自愿留下的。 |
| **地脉结晶** | 大地能量脉络中凝结的坚固结晶。在世界撕裂后仍然沉默地生长。 |
| **远古铭文石** | 刻有失落知识的石碑碎片。有一些字，墨语认识。更多——他不认识。 |

### 2.4 传奇材料（1种）

| 材料名称 | 描述文案 |
|---------|---------|
| **悖时薄片** | 凝固的、记录时间悖论的时空片段。世界撕裂的瞬间，时间在这里折叠——过去和未来在薄片的两面相望，永远触不到彼此。 |

## 三、灵魂之核图鉴（敌人背景描述）

> **设计说明**：为3类敌人大类撰写图鉴文本，每条30-50字。文本记录于墨语的炼金笔记"灵魂之核"图鉴页中。

### 3.1 晶化原住民

> 被晶侵蚀的世界原住民。在世界碎裂的那一刻，晶包裹了他们——连同他们的记忆、恐惧、和最后一声叹息。他们的眼睛里有晶的光芒，但那光芒是借来的。击败他们不是杀戮，是释放。每一次倒下，都有一缕不属于晶的意识从碎片中升起，然后消散——像营火熄灭后的最后一缕烟。

### 3.2 墨语负面实体

> 墨语暴走撕裂世界时，从灵魂裂隙中涌出的负面情绪具现化——愤怒、恐惧、绝望、执念。它们是墨语的一部分，是他不敢面对的影子。在世界碎片中游荡，寻找它们的"本体"——那个它们既恨又无法离开的源头。击败它们，不是消灭——是重新接纳。每一次交锋，都是墨语与自己的对话。他不说话。它们替他说。

### 3.3 S-SN试炼生物

> **解锁条件**：仅在遭遇S-SN试炼生物后解锁。此条目在游戏早期不可见，墨语此时尚未知晓S-SN的存在和编号099宇宙的真相。
>
> 来自编号099宇宙的造物，被S-SN投放到世界碎片中。它们的构造不属于这个世界——晶质的骨骼、绯红色的能量核心、眼中没有敌意只有观测。它们不主动攻击——它们等待。等待墨语走到它们面前，举起剑，证明自己配得上继续前进。每一次战斗，都在S-SN的观测日志上留下一条记录。不是测试——是注视。她在注视。

## 四、设计说明

| 类型 | 数量 | 设计要点 |
|------|------|---------|
| 炼金配方背景 | 14条 | 每条30-50字，暗示配方背后的故事或含有该配方信息的人。基础配方聚焦创之人/伊卡洛斯/绯红家族，进阶配方拓展到碎片世界的幸存者与历史遗产，传奇配方融入编号099宇宙、悖时薄片等核心设定。叙事风格为"墨语在炼金笔记中的记录"，语气克制但有温度。 |
| 材料描述 | 24条 | 每条15-25字，精炼且富有世界感。基础材料强调"碎片世界中的沉默存在"，稀有材料加入"时间/记忆/能量"的异常质感，史诗/传奇材料暗示更深层设定（编号099文字、大地脉络）。所有描述避免冷冰冰的"属性说明"，保持文学性温度。 |
| 敌人图鉴 | 3条 | 每条30-50字，为三大敌人类别撰写。晶化原住民强调"释放而非杀戮"，负面实体强调"墨语面对自己"，S-SN试炼生物强调"观测而非敌意"。图鉴以墨语笔记语气撰写——不是客观的怪物图鉴，是他对敌人的理解。 |
