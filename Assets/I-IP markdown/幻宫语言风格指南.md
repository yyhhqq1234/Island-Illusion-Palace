# 幻宫：时空回响 -- 语言风格指南

## 一、总体基调

《幻宫：时空回响》的语言风格以 **"克制的悲怆"** 为核心定位。游戏讲述的是一个关于失去、执念与救赎的故事，但叙事语言必须避免陷入滥情或说教。整体追求"句式简洁但情感厚重"——用最少的词传递最复杂的情绪。

### 基调关键词
- **内敛**：情绪不直白宣告，而是通过细节与留白传达
- **古典**：句式偏向书面语，但不过度文白夹杂
- **沉重**：不回避痛苦的重量，但也不沉溺于痛苦
- **温度**：在最暗的时刻保留一丝温热——那是莎娜留下的光

---

## 二、角色语体

### 2.1 墨语语体

**定位**：内敛深沉，偏古典。独白使用第一人称。

**语体特征**：
- 句式简洁，不堆砌修辞。墨语不是诗人，他是经历过太多、不愿多言的战士和工匠。
- 用词偏书面语，但避免生僻古语。参考标准：受过良好教育的青年贵族，在孤独中养成了与自己对话的习惯。
- 多使用短句和断句，体现他在思考时的停顿与自我审视。
- 偶尔用锻造/炼金/剑术的比喻，这是他人生经历的自然投射。

**语气范围**：
- 低负担（冷静反思）：克制、理性，带有一丝自我怀疑
- 中负担（矛盾加剧）：句子变短，断句增多，出现重复与自我纠正
- 高负担（偏执）：句子更加破碎，逻辑跳跃，甚至出现"对自己说话"的喃喃自语

**禁用**：
- 现代口语化表达（"好吧""算了""随便"）
- 过度浪漫化的比喻（"她的眼睛像星辰"——这不是墨语会说的话）
- 直白宣言（"我一定要救她！"——墨语不会如此宣告，他会说"必须继续走"）

### 2.2 莎娜语体（记忆碎片动画中）

**定位**：外柔内刚，比墨语更直接，但同样偏古典。

**语体特征**：
- 比墨语多一分温度，少一分沉重。她是墨语世界中的光。
- 在温情时刻可以直白表达情感，但始终保有一份绯红家族继承者的矜持。
- 涉及战斗与责任时，语气果断，不拖泥带水。

### 2.3 S-SN语体（最终战对话）

**定位**：神性与人性交融，语气复杂多层。

**语体特征**：
- 第一阶段（测试）：冷峻、疏离，带审视意味
- 第二阶段（混乱）：语句片段化，多视角交织
- 第三阶段（全力）：威严、压倒性，但隐藏着犹豫
- 第四阶段（揭示）：温柔但悲伤，像跨越了无数宇宙后终于说出的真心话

**关键原则**：S-SN不直接说"我爱你"或"你要成长"。她通过提问、反问和不完整的句子来传递信息——因为她自己也未必能完全理清自己的动机。

## 三、UI语体

**定位**：简洁直接，使用第二人称。

**语体特征**：
- 所有UI文本（按钮、提示、系统说明）使用第二人称"你"。
- 追求"一眼可读"——玩家在激烈战斗中不需要花时间理解文字。
- 功能性优先，情感色彩降到最低，但保留温度。
- 使用标准中文，避免网络用语。

**格式规范**：
- 按钮文本：2-4字为佳（"确认""返回""进入幻宫"）
- 提示文本：30字以内为佳
- 教程文本：分步呈现，每步一句话
- 炼金配方/物品描述：15-25字，可带有世界观风味

## 四、禁用词汇列表

以下词汇及表达方式在《幻宫》中**禁止使用**，以确保语言风格的一致性：

### 4.1 现代/网络用语
- 网络流行语（"绝绝子""yyds""破防""上头"）
- 现代口语填充词（"嘛""啦""咯""哦"作为句末语气词）
- 英文缩写混入中文（"OK""get到了""bug"）
- 二次元萌系表达（"~""的说""喵"）

### 4.2 过度直白的情感表达
- "我爱你"（墨语和莎娜不会直接说这句话——他们的情感通过行动和细节传达）
- 直白的自我鼓励（"我能行！""加油！"）
- 煽情式宣告（"为了她，我什么都愿意做！"）

### 4.3 出戏词汇
- 现代科技词汇（"数据""程序""下载""系统"——除非在炼金笔记的特定语境中）
- 现代政治词汇（"民主""人权""自由"——ISLAND世界有自己的政治框架）
- 现实世界专有名词（"中国""地球""人类"——应使用"这片大陆""世界""世人"）

### 4.4 风格不符词汇
- 过度文白夹杂（"余""吾""汝""之乎者也"——墨语不是古代人）
- 轻浮表达（"小意思""随便玩玩""垃圾"）

## 五、多周目元叙事文本风格

**定位**：**"朦胧的既视感"**，而非"直白的破壁"。

**核心原则**：
- 墨语永远不会说"我好像在一个游戏里"或"这一切都是被设计的"。
- 元叙事信息通过**模糊的既视感**传达：墨语在某些时刻感到"这个场景我好像经历过"、"这个选择我好像做过"。
- 多周目文本的差异应该是"同一个场景的不同角度"，而非"上一次是假的，这一次是真的"。

**具体手法**：

| 周目 | 元叙事手法 | 示例 |
|------|-----------|------|
| 1周目 | 无元叙事，所有内容以"首次体验"语气呈现 | 营火独白："又一片……莎娜，你到底留下了多少碎片？" |
| 2周目 | 微妙既视感，墨语对某些场景产生"莫名的熟悉" | 营火独白："……奇怪。这个营火，我好像来过。不——怎么可能，我不可能来过这里。" |
| 3周目 | 既视感增强，开始出现"选择的重量" | 最终战前："如果……如果这一切已经发生过，我还会做同样的选择吗？" |
| 4周目+ | 墨语不再惊讶于既视感，而是将其视为"契约的代价" | 营火独白："又是这个梦。我已经不会问'为什么'了。契约在提醒我——每一次选择，都在我灵魂上刻下痕迹。" |

**严禁**：
- "这一次我一定要……"（暗示知道上一次失败）
- 第四面墙破裂（"有人在看着我"）
- 时间循环的直白宣告（"我又回到了原点"）

## 六、示例文本

### 6.1 墨语独白示例

**示例1（低负担，收集首个碎片后）**：
> 原来她小时候是这样的。那么小，就已经能握住剑了。我六岁时在做什么？在创之人的锻造房里砸核桃。她大概不知道我这段黑历史。

**示例2（中负担，多次死亡后）**：
> 第三次了。契约在保护我……但每一次回来，我都觉得有什么东西被留在了黑暗里。不重要。也许是我不需要的东西。

### 6.2 UI提示示例

**示例1（首次进入营火）**：
> 营火：在此休息可恢复生命与负担。墨语会在火光中陷入沉思。

**示例2（激活记忆碎片）**：
> 激活记忆碎片将获得共鸣增益，但会增加负担。你最多可同时激活3个碎片。

### 6.3 死亡提示示例

**示例1（初次死亡）**：
> 黑暗中，你听到了她的声音。契约的光芒将你拉回营火旁——但每一次归来，都让灵魂更沉重一分。

**示例2（多次死亡后）**：
> 你又回来了。她还在呼唤你。但你是否注意到——每死一次，那声音就远了一点？

### 6.4 加载画面提示语示例

**示例1（世界观提示）**：
> 死灵圣王之眼不是诅咒，是选择。墨语在创之人的锻造房里学到的第一课：锻造就是选择——保留什么，舍弃什么。

**示例2（系统提示）**：
> 负担超过70时，你将听到不该听到的声音。那是契约在测试你的极限——或者，是你在测试契约的耐心。

## 七、写作检查清单

在提交任何叙事文本前，请逐条确认：

1. [ ] 墨语独白是否使用第一人称？
2. [ ] 句式是否简洁（单句不超过30字为佳）？
3. [ ] 是否避免了禁用词汇列表中的所有词汇？
4. [ ] 情感表达是否足够克制（"展示"而非"宣告"）？
5. [ ] UI文本是否使用第二人称"你"？
6. [ ] 多周目文本是否遵循"朦胧既视感"原则（无直白破壁）？
7. [ ] 是否与ISLAND系列设定保持一致（编号100宇宙、墨语身份、S-SN来源）？
8. [ ] 是否考虑了该文本所处的负担区间对应的语体变化？
