# 暂停菜单系统 - 快速开始指南

## 3步让暂停菜单工作起来

### 第1步：创建暂停菜单
在Unity编辑器中：
1. 打开任意场景
2. 菜单栏选择：`GameObject > IIP > Create Compact Pause Menu`
3. 等待几秒钟，系统会自动生成完整的暂停菜单

### 第2步：创建预制件
1. 在Hierarchy中选中新创建的 `PauseMenu` 对象
2. 将其拖拽到 `Assets/Prefab/` 文件夹
3. Unity会询问是否创建预制件，选择 `Create Original Prefab`

### 第3步：添加到游戏中
方法A - 添加到Bootstrap（推荐）：
1. 找到场景中的 `[IIP] Bootstrap` 对象
2. 添加 `PauseMenuInitializer` 组件
3. 将刚才创建的 `PauseMenu` 预制件拖到 `Pause Menu Prefab` 字段

方法B - 直接放置在场景中：
1. 将 `PauseMenu` 预制件拖拽到每个游戏场景中

## 测试

1. 点击Unity的播放按钮
2. 按 `ESC` 键
3. 应该会看到居中的暂停菜单
4. 点击"继续游戏"恢复

## 常见问题

**Q: 菜单太大/太小怎么办？**
A: 选中PauseMenu对象，在Inspector中调整CanvasScaler的Reference Resolution或Match值

**Q: 如何修改颜色？**
A: 在Hierarchy中展开PauseMenu，选中PausePanel或按钮，修改Image组件的颜色

**Q: 如何添加背景图片？**
A: 选中PausePanel，将Sprite拖到Image组件的Source Image字段

**Q: 设置按钮不工作？**
A: 需要创建一个设置面板预制件，然后在PauseMenu脚本的Inspector中配置settingsPanelPrefab字段
