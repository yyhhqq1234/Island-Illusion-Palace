# 幻宫：时空回响 - 暂停菜单系统重设计说明

## 概述

本次重设计解决了原暂停菜单布局不合理、超出屏幕范围的问题，创建了一个紧凑、居中的暂停菜单UI系统。

## 主要改进

### 1. 紧凑的UI布局
- **面板尺寸**：450×550 像素（比原设计更紧凑）
- **垂直间距**：20 像素
- **内边距**：20 像素

### 2. 居中定位
- 所有UI元素使用 `anchorMin` 和 `anchorMax` 为 (0.5, 0.5)
- 面板会始终显示在屏幕中央，不受分辨率影响

### 3. CanvasScaler 自适应配置
```csharp
UI Scale Mode: Scale With Screen Size
Reference Resolution: 1920×1080
Screen Match Mode: Match Width Or Height
Match: 0.5 (在宽和高之间平衡)
```

### 4. 深色调配色方案
- **面板背景**：(0.08, 0.08, 0.12, 0.95) - 深紫色调
- **按钮背景**：(0.15, 0.15, 0.2, 0.9)
- **遮罩背景**：(0, 0, 0, 0.7)
- **文本颜色**：白色/淡紫色

## 使用方法

### 在Unity编辑器中创建暂停菜单

1. 打开Unity编辑器
2. 确保项目已打开
3. 在菜单栏选择：`GameObject > IIP > Create Compact Pause Menu`
4. 系统会自动生成完整的暂停菜单系统

### 集成到游戏中

#### 方法1：作为独立对象（推荐）
1. 使用上述方法创建暂停菜单
2. 将 `PauseMenu` 对象拖拽到 `Assets/Prefab/` 文件夹创建预制件
3. 在每个游戏场景中实例化该预制件

#### 方法2：挂载到GameManager
1. 创建暂停菜单后
2. 将其设置为 `GameManager` 的子对象
3. 确保 `DontDestroyOnLoad` 功能正常工作

### 功能说明

PauseMenu.cs 脚本保留了所有原有功能：

1. **ESC键切换**：按ESC键可暂停/继续游戏
2. **继续游戏**：恢复游戏运行
3. **设置**：打开设置面板（需要配置settingsPanelPrefab）
4. **返回主菜单**：带确认对话框返回主菜单
5. **退出游戏**：带确认对话框退出游戏
6. **动画效果**：淡入淡出+缩放动画
7. **DontDestroyOnLoad**：跨场景保持

## 对象结构

```
PauseMenu (根对象)
├── Canvas
├── CanvasScaler (自适应)
├── GraphicRaycaster
├── PauseMenu (脚本)
├── DimMask (屏幕遮罩)
├── PausePanel (主面板)
│   └── Content (内容容器)
│       ├── Title (标题文本)
│       ├── ResumeButton (继续游戏按钮)
│       ├── SettingsButton (设置按钮)
│       ├── MainMenuButton (返回主菜单按钮)
│       └── QuitButton (退出游戏按钮)
└── ConfirmDialog (确认对话框)
    ├── DialogText (对话框文本)
    └── ButtonGroup (按钮组)
        ├── ConfirmButton (确认按钮)
        └── CancelButton (取消按钮)
```

## 配置说明

### PauseMenu脚本的Inspector配置

| 字段 | 说明 | 默认值 |
|------|------|--------|
| hideOnStart | 启动时隐藏菜单 | true |
| useESCKey | 使用ESC键切换 | true |
| dimScreen | 暂停时调暗屏幕 | true |
| dimOpacity | 遮罩透明度 | 0.7 |
| pausePanel | 暂停菜单面板 | 自动配置 |
| dimMask | 屏幕遮罩图片 | 自动配置 |
| resumeButton | 继续游戏按钮 | 自动配置 |
| settingsButton | 设置按钮 | 自动配置 |
| mainMenuButton | 返回主菜单按钮 | 自动配置 |
| quitButton | 退出游戏按钮 | 自动配置 |
| confirmDialogPanel | 确认对话框面板 | 自动配置 |
| dialogText | 对话框文本 | 自动配置 |
| confirmButton | 确认按钮 | 自动配置 |
| cancelButton | 取消按钮 | 自动配置 |
| settingsPanelPrefab | 设置面板预制件 | 需手动配置 |

## 测试验证

在Unity编辑器中测试：

1. 点击播放按钮
2. 按ESC键 - 应该显示暂停菜单
3. 点击"继续游戏" - 应该关闭菜单并恢复游戏
4. 尝试不同的分辨率 - 菜单应该始终居中且完整显示

## 注意事项

1. **设置面板**：当前 `settingsPanelPrefab` 字段需要手动配置
2. **字体**：使用Unity内置的LegacyRuntime字体，可根据需要替换
3. **美术资源**：可添加背景图片、按钮图片等美术资源增强视觉效果
4. **IIPBootstrap集成**：确保PauseMenu与IIPBootstrap系统兼容
