**《幻宫：时空回响》整合优化策划案**

**一、项目总览**

-   **游戏名称**：幻宫：时空回响（Island: Illusion Palace）

-   **类型**：2D Roguelite动作RPG + 策略炼金 + 灵魂召唤

-   **世界观**：ISLAND1结局if线，莎娜死亡后墨语力量暴走形成"幻宫"，需收集其灵魂碎片复活

-   **核心玩法**：即时战斗 + 灵魂召唤（冷却制）+ 弱点属性 + 炼金合成 +
    记忆碎片叙事

-   **平台**：PC（Steam）优先

-   **目标**：多结局、多周目、Build多样性

**二、核心系统摘要**

**1. 灵魂召唤系统（冷却制）**

-   **召唤物来源**：击败敌人掉落灵魂之核（白/蓝/金）

-   **召唤方式**：R键轮盘或G键快速召唤，Alt键召回全部

-   **召唤限制**：3个出战位，全局冷却2秒

-   **召唤物类型**：临时（战斗结束消失）与永久（可升级、突破）

-   **记忆碎片共鸣**：最多激活3个碎片，提供战斗增益，每个增加负担

**2. 战斗系统**

-   **武器类型**：剑、法杖、镰刀、晶能武装（各有元素属性）

-   **弱点系统**：7种属性（冰、火、雷、水、灵魂、圣光、物理），弱点伤害×1.8，抗性伤害×0.4

-   **武器升级**：+5系统，每级提升伤害，+3/+5解锁特效槽

-   **闪避与奔跑**：空格闪避（无敌帧），Shift奔跑

-   **武器晶能变体（Crystal Aspect）系统**：
    -   每类武器拥有2-3种晶能变体，改变攻击模式/手感/策略，不改变基础数值框架
    -   变体通过嵌入"晶能核心"激活，晶能核心可在时空裂隙或BOSS掉落中获取
    -   变体之间可自由切换（需在营火处操作），无需消耗资源
    -   **剑类变体示例**：

**3. 炼金系统（探索式合成）**

-   **材料**：24种（基础/稀有/史诗/传奇），价值1-40

-   **配方**：14个（4基础+10隐藏），需玩家探索组合

-   **合成逻辑**：优先匹配配方，次为材料合成，失败按素数分解返还

-   **成功率**：受智慧、负担、记忆碎片影响

**4. 属性与成长**

-   **四大属性**：力量（物伤/负重）、体质（生命/负担抗性）、智慧（灵魂控制/炼金/魔法）、共鸣（碎片发现率）

-   **升级**：每级2属性点，上限50（可突破至60）

-   **Build流派**：物理战士、魔法使者、召唤大师、均衡探索者

-   **永久成长**：多周目继承最多30点属性

**5. 负担系统**

-   **来源**：使用灵魂法术、收集之核、激活记忆碎片

-   **影响**：\>70降低属性与炼金成功率，100强制传送至营火

-   **恢复**：营火休息、炼金药剂

-   **事件解耦设计**：
    -   负担系统仅负责负担值增减与阈值判定，不直接修改其他系统状态
    -   所有状态变更通过 `GlobalEventManager` 广播事件，消费方自行订阅
    -   **BurdenLevel 枚举定义**：

    -----------------------------------------------------------------------
    等级        数值区间       名称          含义
    ----------  -------------- ------------- --------------------------------
    0           0-29           Clear         负担清朗，无负面影响

    1           30-49          Normal        负担正常，处于安全范围

    2           50-69          Elevated      负担升高，轻微风险预警

    3           70-89          High          负担高危，属性/炼金成功率下降

    4           90-100         Critical      负担临界，强制传送风险迫近
    -----------------------------------------------------------------------

    -   **新增事件**：`OnBurdenLevelChanged(BurdenLevel oldLevel, BurdenLevel newLevel)` — 负担等级变化时广播，携带旧/新等级参数
    -   各消费方系统（战斗、炼金、地图、音频等）自行订阅该事件，根据等级变化决定响应行为，实现系统间零硬耦合

**6. 地图与探索**

-   **地图类型**：自然地图（7种）、人文地图（3种）、特殊区域（实验室/记忆区域）、最终区域

-   **生成方式**：随机区块组合（资源/战斗/安全区块）

-   **连接机制**：时空门（稳定）、时空裂隙（随机，5%概率）

-   **多周目**：地图布局、敌人组合、资源分布重置

-   **时空门奖励预览机制**：
    -   玩家与时空门交互时，在传送确认界面展示以下预览信息：
    -   **目标区域类型标签**：显示该门通往的地图类型（如"荒原""废墟都市"等）
    -   **预估难度等级**：基于玩家当前属性与目标区域敌人配置，显示"简单/适中/困难/极难"
    -   **1个高亮信息**：随机展示一项目标区域的特色（如"稀有材料丰富""精英敌人密集""含记忆区域入口"），每次生成时随机，不可刷新
    -   设计意图：提供决策依据但保留探索惊喜，P0核心体验闭环

-   **营火自动存档机制**：
    -   **存档性质**：营火存档为一次性存档——玩家读取该存档后，存档立即被删除，不可重复使用
    -   **自动触发**：玩家退出游戏（关闭窗口/Alt+F4/返回主菜单）时，自动在最近访问过的营火处生成存档
    -   **断点续玩**：确保玩家可从中断处继续，同时防止存档反复读取（Save Scumming）
    -   设计意图：Roguelite容错与公平性平衡

**7. 叙事与结局**

-   **记忆碎片**：7个（童年、战斗、情感、知识、牺牲、希望、真理），收集后触发记忆回放

-   **结局条件**：取决于核心碎片数、记忆碎片数、最终负担值、最终战选择

-   **四大结局**：完美重构（真结局）、残缺回归、力量反噬、牺牲救赎

**8. 资源与经济**

-   **三大资源**：灵魂（升级/强化）、灵魂精华（突破/修复）、记忆碎片（剧情/共鸣）

-   **获取**：杀敌、采集、任务、炼金

-   **消耗**：角色升级、武器强化、召唤物突破、炼金合成

-   **循环**：战斗→资源→强化→更高难度挑战

**9. 敌人与难度**

-   **敌人分类**：晶化生物、灵魂残留、科技造物、记忆具现

-   **难度模式**：简单、普通、困难、疯狂（真结局需疯狂）

-   **动态难度**：根据连战胜率、负担值调整

-   **防挫败**：连败降低难度、保底掉落、检查点系统

**三、操作与界面（PC）**

-   **移动**：WASD/方向键

-   **攻击**：鼠标左键（普通）、右键（特殊）

-   **闪避**：空格键

-   **召唤**：R键（轮盘）、G键（快速）、Alt键（召回）

-   **物品**：数字键1/2/3

-   **菜单**：C角色、I背包、M地图、J日志、Esc系统

**四、开发要点**

-   **团队**：3-5人核心

-   **周期**：12-16个月

-   **美术**：2D卡通，蓝紫灵魂主题

-   **音效**：动态音乐，随负担/战斗状态变化

-   **目标**：20-25小时主线，40+小时全收集，多结局重玩

**五、设计哲学**

-   **简化操作**：冷却制召唤降低上手门槛

-   **深度策略**：弱点系统、记忆配置、负担管理提供策略选择

-   **叙事融合**：碎片收集推进剧情，结局反映玩家选择

-   **自洽世界观**：所有机制均有ISLAND设定支持

-   **重玩驱动**：多周目变化、Build多样性、隐藏内容

**六、清单摘要（详见各清单文件）**

-   **材料**：24种，价值1-40

-   **敌人**：8普通+7精英/BOSS

-   **武器**：15把，4类×7元素

-   **召唤物**：8普通+7永久+1炼金创造

-   **配方**：14个（4基础+7进阶+3传奇）

**八、核心循环与单局流程验证**

**8.1 核心循环验证：探索 → 守护者 → 时空门**

**验证目标**：确保游戏最基本的驱动循环清晰、可重复且富有激励。

![](media/image1.png){width="5.097222222222222in"
height="7.479068241469816in"}

**循环验证说明**：

1.  **驱动力明确**：击败"时空守护者"是解锁进程的核心目标，提供清晰的短期目标。

2.  **决策点丰富**：时空门提供多个区域选择，玩家需根据自身状态（负担、资源、Build）决定路线，策略性显著。

3.  **支持系统完美嵌入**：

    -   **探索**：为战斗与收集提供场景。

    -   **战斗与召唤**：是击败守护者的核心手段。

    -   **炼金与资源**：为挑战守护者提供战前准备与战后恢复支持。

    -   **负担系统**：高风险高回报的决策调节器（如是否在负担时挑战）。

4.  **Roguelite循环成立**：单局内此循环重复4-6次（对应主要区域），多周目因地图、敌人、连接随机化而保持新鲜感。

**8.2 单局游戏流程验证**

**验证目标**：确保单局游戏体验节奏张弛有度，新手引导平滑，核心系统逐层解锁，并在终局形成合力。

  ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
  阶段                 游戏时间      核心目标                                                 系统引入/运用                                                                关键节点与验证点
  -------------------- ------------- -------------------------------------------------------- ---------------------------------------------------------------------------- ---------------------------------------------------------------------------------------------------------------------------------------------------
  **开局与定位**       0-10分钟      熟悉操作，获得首个灵魂之核。                             移动、攻击、闪避教学；首次召唤教学。                                         **验证**：玩家在安全环境下理解基础操作，并获得第一个召唤物（临时）。

  **首个区域探索**     10-40分钟     探索初始区域，收集基础资源，击败首个**时空守护者**。     炼金系统（合成治疗药剂）；负担系统初识；地图探索机制。                       **验证**：玩家通过探索与简单战斗完成资源积累，成功挑战首个BOSS，解锁**时空门**，获得明确的正反馈。

  **中期建设与抉择**   40-90分钟     探索2-3个新区域，构建核心Build，收集首个**记忆碎片**。   武器强化+3；永久召唤物获取与升级；记忆碎片共鸣配置；负担管理成为重要考量。   **验证**：玩家Build初步成型，理解属性、装备、召唤物的协同。首次**记忆区域**挑战提供叙事高潮与强力碎片奖励，负担值成为风险决策的核心。

  **终局准备与决战**   90-120分钟+   进入最终区域，挑战**Scarlet Soul Shana (S-SN)**。        所有系统协同：最优记忆配置、高级炼金药剂、满级主力召唤物、负担精准控制。     **验证**：前期所有积累（碎片数量、负担值、Build强度）直接影响最终战难度与**结局走向**。战斗是多系统理解的终极测试，结局提供情感与游戏性双重回报。
  ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

**单局流程验证结论**：

1.  **节奏合理**：每阶段都有明确目标与成就感节点（首个守护者、首个记忆碎片、最终战）。

2.  **引导渐进**：系统复杂度随玩家成长逐步展开，避免信息过载。

3.  **系统聚合**：前期分散的系统（召唤、炼金、负担）在终局决策中紧密结合，验证了系统设计的深度与关联性。

4.  **重玩性基础稳固**：单局流程的紧凑体验（\~2小时）非常适合Roguelite的"再来一局"心流，而多结局与多周目变化提供了重复游玩的长期目标。
