## 〇、视觉核心规范速查

### 0.1 武器设计哲学

- **暗色手绘基底**：武器主体为深色金属、骨质、晶化材质，避免纯黑(`#000000`)，最深的金属色使用 `#1A1A2E`
- **晶化侵蚀统一语言**：所有武器在强化 +3 后出现晶化生长，晶簇从握柄向刃尖蔓延
- **灵魂辉光点缀**：元素附魔效果为武器上唯一的"自发光"元素
- **类型辨识度**：四类武器在轮廓上有明确差异——剑(细长单刃)、法杖(长柄+顶端聚焦)、镰刀(弯刃长柄)、晶能武装(短小多段)

## 一、武器类型视觉定义

### 1.1 剑类 (Sword)

| 属性 | 规格 |
|------|------|
| **轮廓特征** | 细长直刃，刃宽 4-6px，总长 64-72px；十字护手宽 18-24px |
| **握持位置** | 护手下方 16px 处，握柄长 20-24px |

## 六、强化升级视觉进程

### 6.1 +0 到 +5 视觉变化

| 等级 | 视觉效果 |
|------|---------|
| +0 | 基础外观 |
| +1 | 刃/杖身增加微弱的边缘高光线 (1px, `#C0C0C0`, 30%) |
| +2 | 高光线变为微光 (`#448AFF`, 50%)，刃面增加微细的晶化蚀刻纹 |
| +3 | 首次可见晶化生长——小簇紫色晶体从握柄处冒出 (4-8px)，特效槽 1 解锁，槽位显示空晶化槽 |
| +4 | 晶化生长延伸至刃身中部，粒子增加 (+2)，晶化簇增大 |
| +5 | 晶化到达刃尖，全面紫色辉光爆发，特效槽 2 解锁，武器获得"完整形态"专属粒子效果 |

### 6.2 升级闪光动画

- 3-5 帧 @10FPS
- 帧 1：武器正常 → 帧 2：整体亮度骤升 200% → 帧 3：白色闪光覆盖 80% → 帧 4：闪光消退 → 帧 5：新等级外观显现
- 伴随晶化生长粒子爆发（由中心向四周扩散，20px 半径，0.3s）
