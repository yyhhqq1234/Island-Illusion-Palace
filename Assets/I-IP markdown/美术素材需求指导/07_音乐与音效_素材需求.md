 ## 〇、听觉设计哲学
 
 ### 0.1 核心理念
 
 - **灵魂是声音的光**：蓝紫色灵魂能量在听觉上对应空灵 synth 与高频泛音，是画面中唯一"自发光"的声源
 - **负担是心跳的重量**：灵魂负担等级通过节奏密度、低频厚度、不协和音程递进，让玩家"听"到压迫感的上升
 - **晶化是金属的冷**：晶(HYC)侵蚀在听觉上表现为金属打击乐、玻璃碎裂质感、频率失真
 - **沉默是叙事的工具**：关键剧情节点（记忆碎片共鸣、Boss死亡瞬间、结局触发）使用刻意留白，让情绪沉淀
 
 ### 0.2 三音源架构
 
 | 音源 | 用途 | 混响 | 优先级 |
 |------|------|------|--------|
 | `musicSource` | BGM 背景音乐，loop=true | 低频切除 200Hz 以下 | 基础层 |
 | `sfxSource` | 游戏世界音效（战斗/召唤/炼金） | 根据场景空间混响 | 中 |
 | `uiSource` | UI 界面音效（点击/确认/选择） | 干信号无混响 | 高 |
 
 ### 0.3 音乐状态优先级
 
 ```
 Silence(0) < MainMenu(10) < Exploration(20) < Battle(50) < BossBattle(100) = Camp(100)
 ```
 - 高优先级状态覆盖低优先级
 - BossBattle 与 Camp 同级最高（游戏逻辑上不会同时出现）
 - 退出高层状态后自动回退到次高优先级
 
 ### 0.4 负担等级与音乐联动
 
 | 负担值 | 等级 | 情绪关键词 | 节奏 | 乐器重点 |
 |--------|------|-----------|------|---------|
 | 0-30 | Low | 希望、神秘、探索 | 缓慢→中等 | 弦乐为主 + 空灵 synth |
 | 31-60 | Medium | 压抑、紧张、不安 | 中等加快 | 铜管加入 + 鼓点加重 |
 | 61-100 | High | 恐惧、压迫、危险 | 快速激烈 | 金属打击乐 + 低频 bass |

 ### 1.1 主界面主题曲 (MainMenu)
 
 | 参数 | 规格 |
 |------|------|
 | 文件名 | `MainTheme.mp3` |
 | 时长 | 2:30 - 3:30 (循环友好) |
 | 格式 | MP3, 192kbps, 44.1kHz, Stereo |
 | 循环 | 无缝循环 (loop=true) |
 | 音量基准 | -12 dBFS (RMS) |
 
 **音乐设计**：
 - **主题**：幻宫世界、时空回响
 - **风格**：空灵神秘 + 史诗前奏
 - **情绪**：神秘 → 期待 → 微弱的悲伤底色
 - **乐器**：空灵 synth pad + 轻微弦乐 + 神秘氛围 + 远处女声哼唱
 - **BPM**：70-80 (缓慢庄重)
 - **调性**：D 小调（神秘感）
 
 **结构建议**：
 - 0:00-0:30 intro: 单音 synth pad 渐入 + 风声氛围
 - 0:30-1:15 A段: 主旋律弦乐 + 轻盈打击
 - 1:15-2:00 B段: 女声哼唱 + 情绪升华
 - 2:00-2:30 尾声: 回归 intro 的空灵，准备无缝循环

 ## 十二、技术规格
 
 ### 12.1 音频格式规范
 
 | 类型 | 格式 | 采样率 | 声道 | 码率 | 说明 |
 |------|------|--------|------|------|------|
 | BGM 背景音乐 | MP3 | 44.1kHz | Stereo | 192kbps | 无缝循环，文件 < 5MB |
 | SFX 游戏音效 | WAV/MP3 | 44.1kHz | Mono | 128kbps | 短促音效，Mono 利于空间定位 |
 | UI 音效 | WAV/MP3 | 44.1kHz | Mono | 128kbps | 干信号，无混响 |
 | 环境氛围 | MP3 | 44.1kHz | Stereo | 160kbps | 循环，低频为主 |
 
 ### 12.2 音量规格
 
 | 类型 | 峰值 (dBFS) | RMS (dBFS) | 说明 |
 |------|------------|-----------|------|
 | BGM | -3 | -12 | 留 headroom 给 SFX |
 | SFX | -1 | -10 | 短瞬态允许接近峰值 |
 | UI | -3 | -15 | 不干扰 BGM/SFX |
 | 环境氛围 | -6 | -24 | 底层铺垫，不喧宾夺主 |
 
 ### 12.3 淡入淡出规范
 
 | 场景 | 淡出 | 淡入 | 总时长 |
 |------|------|------|--------|
 | 音乐状态切换(BGM) | 0.5s | 0.5s | 1.0s |
 | 场景转场 | 0.3s | 0.7s | 1.0s |
 | 章节切换 | 1.0s | 1.0s | 2.0s |
 
 代码实现：`GameplayAudioManager.FadeMusic()` 协程，`musicFadeTime` 默认 1.0s

 ## 十四、AI 生成工具链参考
 
 本项目音乐素材可通过 AI 音乐生成工具辅助创作，推荐工作流：
 
 | 工具 | 适用类型 | 说明 |
 |------|---------|------|
 | Suno AI | BGM 主题曲/探索曲/战斗曲 | 文本提示词生成，支持风格/乐器/BPM/调性控制 |
 | Udio | 高品质 BGM | 更精细的风格控制，适合情感向曲目 |
 | ElevenLabs | 环境音效/人声采样 | 灵魂低语、记忆闪回人声 |
 | Audacity / Reaper | 后期处理 | 淡入淡出、无缝循环、音量标准化、格式转换 |
 
 **Suno 提示词规范**（参考 [I_IP地图音乐配置指南](../02_数据清单/I_IP地图音乐配置指南.md) "音乐风格建议"章节）：
 - 风格标签：`dark fantasy, orchestral, synth, {地图特有元素}`
 - 情绪标签：`mysterious / tense / fearful / epic / warm`
 - 乐器指定：`strings, synth pad, brass, percussion, {特色乐器}`
 - 结构标签：`intro, build, climax, outro, loop-friendly`
 
 **质量检查清单**：
 - [ ] 无缝循环检查（BGM 首尾衔接无爆音）
 - [ ] 音量标准化（RMS 符合第12.2节规格）
 - [ ] 频谱不冲突（BGM 与 SFX 频段分离）
 - [ ] 情绪一致性（与地图清单环境描述匹配）
 - [ ] 负担等级递进感（低→中→高 压迫感明确递增）
