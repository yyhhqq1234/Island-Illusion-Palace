## 〇、视觉风格核心规范速查

> 本规范为《幻宫：时空回响 — 视觉风格指导手册》的精简速查版。完整规范请参考源文档：`幻宫_视觉风格指导手册.md`。

### 0.1 全局设计理念

| 关键词 | 视觉解读 |
|--------|---------|
| **暗色手绘** | 深色主基调，笔触保留手工质感，最深色 `#1A1A2E`（非纯黑 `#000000`）|
| **晶化侵蚀** | 几何化晶簇（六方柱+菱面体，120度分叉），青蓝 `#00D4FF` → 蓝紫 `#4A3A8C`，五层级 0-5 |
| **灵魂辉光** | 蓝紫色柔光 (`#4A3A8C`~`#7B5EA8`)，脉动呼吸效果，边缘微弱色散 |
| **手绘感优先** | 保留笔触纹理，避免纯渐变，多层叠加而非单层渲染 |

### 0.3 角色辨识度规范

| 原则 | 说明 |
|------|------|
| **轮廓唯一性** | 每个角色类型的剪影必须唯一，32×32像素缩小后仍可区分 |
| **头身比锚定** | 玩家/人形：3-3.5头身（顶视角）|
| **武器/配饰外扩** | 配饰至少占总轮廓面积15% |
| **方向指示** | 角色轮廓隐含面朝方向，0.1秒内判断朝向 |

### 0.4 敌人 vs 召唤物视觉区分

| 属性 | 敌人 | 召唤物（含契约伙伴） |
|------|------|---------------------|
| **主色调** | 保持原色（根据类型）| 叠加蓝紫灵魂光晕 |
| **外框/光晕** | 无额外光晕（仅BOSS有红色警告）| 半透明蓝紫光晕外框 `rgba(74,58,140,0.31)` Additive混合，外扩2-4px，呼吸周期2秒 |
| **脚下标识** | 红色敌对光圈（战斗状态）| 蓝色友方光圈（持续） |
| **透明度** | 100%不透明 | 85-90%半透明 |
| **死亡效果** | 红色崩解粒子 | 蓝紫升腾粒子 + 灵魂之核掉落 |
| **晶化纹理** | 随阶段递增(0-5) | 始终轻度晶化(阶段1-2) |

### 0.5 动画帧数标准

| 动画状态 | 最少帧数 | 播放速度 | 备注 |
|---------|---------|---------|------|
| **Idle（待机）** | 6帧 | 8-10 FPS | 含呼吸感微动 |
| **Walk（行走）** | 6帧 | 10-12 FPS | 匹配移动速度，避免滑步 |
| **Run（奔跑）** | 6帧 | 12-15 FPS | Shift加速时切换 |
| **Attack（攻击）** | 8帧 | 12-15 FPS | 快起慢收：蓄力1/3→挥砍1/3→收招1/3 |
| **Dash（闪避）** | 4帧 | 15-20 FPS | 中间帧可用运动模糊 |
| **Hurt（受击）** | 3帧 | 12 FPS | 短暂闪白+击退 |
| **Death（死亡）** | 8帧 | 8-10 FPS | 崩解为粒子效果衔接 |
| **Summon（召唤）** | 6帧 | 10 FPS | 从地面法阵中升起 |

**方向要求**：人形角色需要 4 方向（down/up/left/right），不可镜像复用。非人形角色 left/right 可镜像复用。

## 一、玩家角色：墨语 (MoYu / MY)

### 1.1 角色概述与设计理念

墨语是玩家直接操控的主角，2D顶视角下的3-3.5头身少年角色。作为死灵圣法神契约继承者，其视觉设计需同时体现三个层次：

1. **表层**：16岁少年的纤瘦身形、简练工装、深黑短发——平实、低调、内敛
2. **契约层**：左手"死灵圣王之眼"戒指的暗金微光、佩剑"叙写者"的蓝紫灵魂辉光——力量标志
3. **负担层**：负担值>70时周身蓝紫光晕颤动、>90时光晕变绯红+晶化纹理浮现——状态的视觉化

**核心设计理念**："暗色中透光的少年"——底色是沉静孤寂的暗（`#2A2A3A`），但灵魂与契约赋予他蓝紫辉光（`#4A3A8C`）和绯红点缀（`#C41E3A`）。他不是传统意义上的"勇者"，而是背负死亡契约的"守望者"。

### 1.2 像素规格

| 属性 | 规格 |
|------|------|
| 角色尺寸 | **64×64 像素** |
| 头身比 | 3-3.5头身（顶视角） |
| 精灵表布局 | 单状态单方向一个精灵表，帧横向排列 |
| 方向数 | 4方向（down / up / left / right），不可镜像复用 |
| 总帧数估算 | 9状态 × 4方向 ≈ 约 200-240 帧（含过渡关键帧） |

#### 1.4.1 Idle（待机）— 6帧 @10FPS

```
┌──────┬──────────────────────────────────────────────────────────────────────────────────┐
│ 帧号  │                              关键帧定义                                          │
├──────┼──────────────────────────────────────────────────────────────────────────────────┤
│ 帧1   │ 呼气末中立位。双脚与肩同宽(间距6px)，双手自然垂于身侧。剑鞘垂直。身体重心在双脚正中。  │
│      │ 胸腔收缩(总高度48px)。头发末梢静止。戒指微光α=30%。持续时间100ms                      │
├──────┼──────────────────────────────────────────────────────────────────────────────────┤
│ 帧2   │ 吸气上升。胸腔微扩(+2px→总高50px)，肩部轻微上提(+1px)。斗篷下摆微扬(+1px)。          │
│      │ 右手剑鞘跟随身体微升(+1px)。左手指环光晕增亮α=40%。头发末梢微动(+1px偏移)。            │
│      │ 持续时间100ms                                                                       │
├──────┼──────────────────────────────────────────────────────────────────────────────────┤
│ 帧3   │ 呼吸高点。胸腔扩张到最大(+3px→总高51px)，肩部微抬(+2px)。斗篷下摆升至最高(+2px)。      │
│      │ 剑鞘微微前倾(1度)。戒指光晕达到峰值α=50%。头发末梢向后飘动(+2px)。                     │
│      │ 持续时间100ms                                                                       │
├──────┼──────────────────────────────────────────────────────────────────────────────────┤
│ 帧4   │ 呼气回落。胸腔回缩至帧2水平(总高50px)。肩部回落(-2px)。斗篷下摆下降中。               │
│      │ 戒指光晕恢复至α=30%。身体重心微后移(0.5px)。头发末梢回落。持续时间100ms                │
├──────┼──────────────────────────────────────────────────────────────────────────────────┤
│ 帧5   │ 中立偏前。身体回到近中立位但微有前倾(前倾1px)。斗篷下摆轻微摆动残留(+1px摆动)。        │
│      │ 剑鞘稳定，戒指光晕正常。胸腔约为帧1水平。头发末梢微有惯性偏移。持续时间100ms             │
├──────┼──────────────────────────────────────────────────────────────────────────────────┤
│ 帧6   │ 微风吹动。头发末梢(+1-2px横向偏移)、斗篷边缘(+1px波动)。身体回到帧1中立位。            │
│      │ 头部微侧(左偏1度)，模拟自然微动。戒指光晕α=30%。然后循环回帧1。持续时间100ms            │
└──────┴──────────────────────────────────────────────────────────────────────────────────┘
```

**Idle特效叠加**：
- 戒指微光：帧1→3亮度爬升，帧3→5亮度下降。使用 Additive 混合的半透明圆光（2px半径）叠加在戒指位置
- 剑鞘宝石：与戒指同步脉动，相位偏移90度（戒指最亮时宝石最暗，反之亦然）
- 周身粒子：负担>70时激活（见1.5节）

- 整体位移变化：每帧角色水平位移约2-3px（与移动速度同步）

```
帧8: 完成。灵魂石回到待机亮度。暗影粒子恢复到3-5个标准轨道。回到Idle
```

##### Summon（被召唤出场）— 6帧 @10FPS

```
帧1: 地面出现暗紫色召唤法阵（直径40px, `#2D1F5E`→`#4A3A8C` 渐变, 扭曲的符文纹理）。法阵内有暗影如烟雾般翻滚
帧2: 法阵激活。暗影烟雾从法阵中大量涌出(如黑色火焰, `#1A1A2E`→`#4A3A8C` 粒子, 15-20个)。烟雾中可见扭曲的灵魂面孔。罗法西亚轮廓以α=10%浮现，先看到白发

帧4: 实体化。α=70%。暗影烟雾开始收缩回法阵。白发从展开状态回落
帧5: 降临完成。α=85%。法阵缩小消失。暗影烟雾散去。灵魂石恢复常规亮度。暗影粒子进入轨道
帧6: 进入Idle。地面残留暗影印记(持续2秒)。进入待机
```

### 3.2 蓝紫光晕实现规范（Additive混合层）

```
层级结构（从底到顶）：
  Layer 0: 角色本体Sprite（α=85-90%）
  Layer 1: 蓝紫光晕叠加层（Additive混合）
           - 颜色: #4A3A8C α=31% (即rgba(74,58,140,0.31))
           - 形状: 角色轮廓的模糊膨胀版本（外扩2-4px）
           - 动画: 透明度正弦波动 sin(t*π) → α范围0.25-0.35
           - 周期: 2秒（与呼吸不同步，形成微妙的不规则感）
  Layer 2: 蓝色友方光圈（SpriteRenderer, 非Additive）
           - 颜色: #4A8CD4 α=40%
           - 形状: 椭圆光环（跟随角色轮廓），线宽2px
           - 位于角色脚下，Sorting Order低于角色
           - 始终显示（不受光晕呼吸影响）
```

### 3.3 召唤物死亡效果规范

```
死亡动画序列（8帧 @10FPS）：
  帧1-2: 召唤物受致命伤（闪白 #FFFFFF α=60%）
  帧3-4: 身体透明化加速（α=70→50%），蓝紫光晕开始急剧增强（α=31→60%）
  帧5-6: 身体崩解（α=50→20%），大量蓝紫粒子从身体涌出
          - 粒子数量: 20-30个
          - 粒子形状: 圆形柔光（非晶化六边形，因为是灵魂而非晶）
          - 粒子颜色: #4A3A8C→#7B5EA8 渐变，2-4px
          - 粒子运动: 上升为主（速度1-2px/帧）+ 随机横向漂移
          - 粒子生命周期: 1-2秒
  帧7-8: 身体完全消失，粒子升腾达到最高（高约40px）
          地面留下"灵魂之核"掉落物：
          - 一个小型发光球体 (#4A3A8C→#00D4FF 渐变, 直径8px)
          - 内部有旋转的灵魂能量（同灵魂石内部纹理）
          - 掉落至地面后微微弹跳1次，然后静止闪烁
          - 闪烁周期: 1秒（亮度在60-100%之间正弦波动）
          - 吸附效果: 玩家靠近时，灵魂之核缓慢飘向玩家（速度2px/帧）
```

### 6.4 设计决策记录

#### A. 为什么墨语是64×64而大部分契约伙伴是48×48/56×56？

墨语作为玩家角色，需要最大的视觉存在感和最丰富的动画细节。64×64的像素空间允许更多的装备细节（戒指、佩剑、斗篷）和更清晰的动画表现。契约伙伴作为召唤物，48×48/56×56在保证辨识度的同时，避免在屏幕上与玩家争抢视觉焦点。

#### B. 为什么罗法西亚是64×64？

罗法西亚在叙事上是最特殊的契约伙伴——她是来自ISLAND系列的跨作品角色，背负诅咒，携带灵魂石。她在游戏中的重要性和视觉复杂程度（白发+灵魂石+暗影拖尾+暗影粒子）需要更大的像素空间来呈现。此外，64×64与玩家同级也暗示了她"特殊"的地位。

#### C. 为什么契约伙伴使用4方向而非allow镜像复用？

契约伙伴作为永久特殊召唤物，其动画品质应与普通临时召唤物拉开差距。4方向不可镜像可确保左右方向的非对称细节（如伊格尼斯的右拳主攻手、苍穹之令的右手权杖、罗法西亚胸前灵魂石）始终正确呈现。普通召唤物（如腐化村民召唤物）可以使用左右镜像复用以节省成本。

#### D. 为什么伊莉丝的Attack被替换为Heal？

伊莉丝是纯粹的治疗型角色，不具备攻击能力。Heal动画替代Attack作为其主要"主动"动画状态，既符合角色定位，又保持了动画状态机的一致性。Heal动画的设计参考了Attack的快起慢收结构，但在视觉效果上完全转向治愈和治疗。

#### E. 为什么苍穹之令的Summon是从天空降下而非地面升起？

苍穹之令是雷神Rustia的化身，雷霆从天空劈下符合"天罚"的叙事意象。这一独特的出场方式也使其在所有契约伙伴中最为醒目，强化了"裁决者"的身份定位。
