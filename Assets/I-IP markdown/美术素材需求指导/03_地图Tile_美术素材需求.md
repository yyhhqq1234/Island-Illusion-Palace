## 一、视觉核心规范

### 1.1 风格基石

| 属性 | 规格 |
|------|------|
| 绘画风格 | 暗色手绘，保留笔触纹理，避免纯渐变填充；多层叠加渲染，非单层 |
| 最深色 | `#1A1A2E`（深蓝紫黑），不可使用纯黑 `#000000` |
| Tile基础尺寸 | **32×32 px**（所有地面Tile统一基准） |
| 变体数量 | 每种地面至少 **5-8 个变体**（形态微差 + 色差 ±5%）避免重复感 |
| 障碍物变体 | 每种障碍物 **2-4 个变体** |
| 装饰物变体 | 每种装饰物 **5+ 个变体** |

### 1.2 晶化矿脉统一规范

所有地图中的晶化矿脉遵循以下几何形态和色彩规则：

```
形态: 六方柱状晶体 + 菱面体尖端，120度夹角分叉（遵循晶体生长规律）
颜色渐变: 深蓝紫(#2D1F5E) → 蓝紫(#4A3A8C) → 青蓝(#00D4FF)
光照: 自发光(Additive混合)，在暗色场景中作为视觉引导
密度: 普通地图中每屏1-3处稀疏点缀，实验室碎片中密集分布

晶簇规格:
  - 小型晶簇: 2-3颗晶体，8-12px
  - 中型晶簇: 4-8颗晶体，16-24px
  - 大型晶簇: 10+颗晶体，32-48px，呈现复杂分叉结构
```

### 1.3 全局色彩锚点

所有地图（无论主色调如何）必须包含以下统一视觉锚点：

| 色彩 | Hex | RGB | 用途 |
|------|-----|-----|------|
| 灵魂蓝紫 | `#4A3A8C` | 74, 58, 140 | 晶化矿脉辉光、灵魂光点、场景氛围粒子 |
| 绯红 | `#C41E3A` | 196, 30, 58 | S-SN叙事线索、危险标识、特定装饰 |
| 晶化青蓝 | `#00D4FF` | 0, 212, 255 | 晶簇尖端高亮、霓虹残光、场景自发光 |
| 暗金 | `#C8A848` | 200, 168, 72 | 神圣遗迹装饰、传说级交互元素 |

### 1.4 8层Tilemap渲染架构

```
Layer -2: 远景层 (Parallax Back)    — 天空底色、远山剪影、大气粒子
Layer -1: 远景装饰层 (Far Back)     — 远景建筑轮廓、大型环境结构
Layer  0: 地面层 (Ground)           — 基础地面Tile(5-8变体)、地板纹理
Layer  1: 地表装饰层 (Ground Deco)  — 裂缝、水渍、晶化纹理、血迹、地面花草
Layer  2: 障碍物层 (Obstacles)      — 墙壁、岩石、树木、建筑结构(含碰撞体)
Layer  3: 场景物件层 (Props)        — 装饰物(营火/宝箱/NPC/可交互物)
Layer  4: 场景前景层 (Foreground)   — 覆盖玩家的前景元素(藤蔓/柱子/拱门)
Layer  5: 光效层 (Lighting Effects) — 自发光辉光、体积光、粒子(Additive混合)
```
