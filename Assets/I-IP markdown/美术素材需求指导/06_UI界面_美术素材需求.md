## 〇、视觉核心规范速查

### 0.1 UI 设计哲学

- **暗色晶化面板**：所有 UI 面板使用深色底 (`#1A1A2E`)，边框以晶化几何线条 (`#4A3A8C`) 勾勒
- **灵魂辉光交互**：可交互元素使用紫蓝辉光 (`#4A3A8C` → `#7B5EA8`) 作为悬停/选中反馈
- **金为尊、红为危**：传奇/稀有作金色 (`#FFD700`)，伤害/危险用绯红 (`#C41E3A`)
- **像素手绘感**：UI 元素保留手绘质感，避免纯矢量感。边框不是纯直线，带有微弱抖动

| 底色 | `#1A1A2E` 85% |
| 角色头像 | 左端 64×64px |
| 名字 | 头像右侧 18pt Bold 角色专属色 |
| 对话文字 | 14pt `#FFFFFF` |
| 继续指示 | 右下角闪烁三角 ▼，3帧 @4FPS |
| 跳过按钮 | 右上角 `Skip >>` 14pt |

### 5.2 通知横幅 (Notification)

| 参数 | 规格 |
|------|------|
| 位置 | 顶部中央下方 40px |
| 最大宽度 | 600px |
| 底色 | `#1A1A2E` 90% |
| 边框 | 左侧色条 4px（类型对应色） |
| 字体 | 16pt White |
| 持续时间 | 3-5s（根据重要性） |
| 入场 | 顶部滑入 0.3s |
| 退场 | 淡出 0.5s |

**通知类型色条**：
- 系统：`#4A3A8C` (紫)
- 战斗：`#C41E3A` (红)
- 奖励：`#FFD700` (金)
- 叙事：`#B0B0C0` (灰)

### 5.3 加载画面 (Loading Screen)

| 参数 | 规格 |
|------|------|
| 背景 | `#1A1A2E` 纯色 |
| 中央元素 | 旋转的晶化碎片 (4帧 @8FPS 旋转动画, 120×120px) |
| 游戏标题 | 底部，24pt `#FFFFFF` |
| 提示文字 | 标题上方，随机显示世界观小知识，16pt `#B0B0C0` |
| 进度条 | 400×6px，`#4A3A8C` 填充 |

### 5.4 存档/读档界面 (Save/Load)

| 参数 | 规格 |
|------|------|
| 存档卡尺寸 | 500×100px |
| 卡内容 | 左：场景缩略图 120×80px / 中：存档信息(时间+地点+进度) / 右：操作按钮 |
| 空卡位 | `#2A2A3A` + "空存档槽" 18pt `#6A6A7A` |
| 选中态 | `#4A3A8C` 边框 3px |

### 5.5 游戏结束/胜利画面

**Game Over (负担=100)**：
- 全屏白色闪光 0.2s → 紫色粒子漩涡 1.0s → 角色被传送到营火旁的动画

**Boss胜利**：
- 全屏金色光波从 Boss 尸体扩散 → "VICTORY" 48pt Bold `#FFD700` 渐显 → 奖励清单弹出
