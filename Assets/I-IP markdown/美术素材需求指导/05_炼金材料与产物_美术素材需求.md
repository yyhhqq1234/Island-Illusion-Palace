## 〇、视觉核心规范速查

### 0.1 炼金视觉哲学

- **材料是"被发现的"**：采集/掉落的材料外观保持自然/原始质感——腐化的残块、结晶的碎片、生物的遗骸——不经过人为美化
- **产物是"被创造的"**：炼金合成后的产物呈现主动设计的形态——精致瓶装、符文封印、有序排列——体现玩家的意图与技巧
- **品质梯度由材质表达**：稀有度不是通过发光强弱区分，而是通过材质本身的价值感——木屑→骨骼→结晶→悖论碎片

### 4.1 资源目录结构

```
Assets/ArtMaterials/Items/Alchemy/
├── materials_basic/         # 10 种基础材料
│   ├── mat_RottenWood.png → mat_WailingVine.png
├── materials_rare/          # 10 种稀有材料
│   ├── mat_MemoryResidue.png → mat_AncientDragonBonePowder.png
├── materials_epic/          # 3 种史诗材料
│   ├── mat_SoulEssence.png
│   ├── mat_LeylineCrystal.png
│   └── mat_AncientRuneStone.png
├── materials_legendary/     # 1 种传奇材料
│   └── mat_ParadoxShard.png
├── products/                # 14 种炼金产物
│   ├── prod_HealingPotion.png → prod_TruthSeerPotion.png
├── ui/                      # 炼金UI专用素材
│   ├── slot_empty.png       # 空槽位
│   ├── slot_material.png    # 材料槽
│   ├── slot_product.png     # 产物槽
│   ├── btn_synthesize.png   # 合成按钮（正常/悬停/按下）
│   ├── effect_success_01~08.png  # 成功VFX序列帧
│   └── effect_fail_01~05.png     # 失败VFX序列帧
└── frames/                  # 边框底板
    ├── frame_common.png
    ├── frame_rare.png
    ├── frame_epic.png
    └── frame_legendary.png
```

### 4.2 文件命名规范

- 材料图标：`mat_{Name}.png`（Name 使用 PascalCase 英语，不含空格）
- 产物图标：`prod_{Name}.png`
- 边框：`frame_{Rarity}.png`
- VFX 序列帧：`effect_{Type}_{帧号}.png`（帧号从 01 开始补零）

### 4.3 着色器要求

| 效果 | 着色器 | 说明 |
|------|--------|------|
| 材料/产物精灵 | Sprite/Default | 标准 |
| 辉光材料（灵魂等） | Sprite/Additive | 叠加发光层 |
| 液体折射 | Sprite/Diffuse | 玻璃瓶透明层 |
| VFX 粒子 | Particles/Additive | 粒子特效 |
