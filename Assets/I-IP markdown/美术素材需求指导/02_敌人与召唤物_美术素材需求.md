## 一、视觉风格核心规范

### 1.1 全局美术锚点

| 锚点 | 规范 | 来源 |
|------|------|------|
| **最深色** | `#1A1A2E`（深蓝紫黑），禁止使用纯黑 `#000000` | 视觉风格指导手册 §1.3 |
| **晶化侵蚀** | 几何晶簇，120度分叉，青蓝 `#00D4FF` → 蓝紫 `#4A3A8C` 渐变，5层级（L0-L5） | 视觉风格指导手册 §7 |
| **灵魂辉光** | 蓝紫色系 `#4A3A8C` ~ `#7B5EA8`，Additive混合，脉动呼吸周期2秒 | 视觉风格指导手册 §1.1 |
| **顶视角** | 2D top-down 正交视角，Z轴仅用于渲染排序 | CLAUDE.md |
| **手绘感** | 保留笔触纹理，避免纯渐变填充，线条带微弱不规则性 | 视觉风格指导手册 §1.1 |

### 1.2 召唤物 vs 敌人视觉区分（绝对规则）

| 属性 | 敌人 | 召唤物 |
|------|------|--------|
| **主色调** | 保持各分类原色 | 叠加蓝紫灵魂光晕 |
| **外框/光晕** | 无额外光晕（仅BOSS有红色警告光晕） | 半透明蓝紫外框 `rgba(74,58,140,0.31)`，Additive混合 |
| **光晕外扩** | — | 2-4像素（按召唤物尺寸比例） |
| **光晕呼吸** | — | 透明度 0.25↔0.35 正弦波动，周期 2 秒 |
| **脚下标识** | 红色敌对光圈（战斗状态） | 蓝色友方光圈（持续） |
| **透明度** | 100% 不透明 | 灵魂状态 85-90% 半透明 |
| **死亡效果** | 红色崩解粒子 | 蓝紫升腾粒子 + 灵魂之核掉落 |
| **晶化纹理** | 随结晶病阶段递增（L0-L5） | 始终为轻度晶化（L1-L2） |

### 1.3 颜色专属规范

| 色系 | Hex范围 | 专属用途 | 禁止用途 |
|------|---------|---------|---------|
| 蓝紫 | `#4A3A8C` ~ `#7B5EA8` | 灵魂能量、召唤物光晕、UI主色 | 非灵魂元素的主色 |
| 青蓝 | `#00D4FF` | 晶化高亮、技能特效、UI选中态 | 非晶元素的纹理 |
| 绯红 | `#C41E3A` | S-SN相关、叙事高潮、致命伤害 | 日常战斗特效 |
| 暗金 | `#C8A848` | 神圣/高等标识、史诗物品 | 普通品质物品 |

### 2.3 敌人动画帧数通用标准

| 状态 | 最少帧数 | 推荐帧数 | 播放速度 | 说明 |
|------|---------|---------|---------|------|
| **Idle** (待机) | 4-6帧 | 6-8帧 | 8 FPS | 含呼吸微动，非完全静止 |
| **Walk** (行走) | 5-6帧 | 6-8帧 | 10-12 FPS | 需与移动速度匹配 |
| **Attack** (攻击) | 5-6帧 | 6-8帧 | 12 FPS | 快起慢收，前1/3蓄力→中1/3挥砍→后1/3收招 |
| **Hurt** (受击) | 3帧 | 4帧 | 12 FPS | 短暂闪白 + 击退 |
| **Death** (死亡) | 5-6帧 | 6-8帧 | 8 FPS | 崩解为粒子效果过渡 |
| **Aggro** (警觉) | 3-5帧 | 4-6帧 | 10 FPS | 发现玩家过渡，从Idle切换到战斗姿态 |
| **Summon** (召唤) | 4帧 | 4-6帧 | 10 FPS | 仅石像鬼等需要，从法阵中升起 |
| **Special** (特殊) | 4-6帧 | — | 8-12 FPS | 如潜行(沼泽潜伏者)、飞行(石像鬼) |

### 2.4 方向要求

| 类型 | 方向数 | 可复用的镜像 |
|------|--------|-------------|
| 人形（腐化村民、骷髅战士、灵魂吞噬者、记忆守护者） | 4方向 (down/up/left/right) | left/right 不可镜像（武器手不同） |
| 四足（结晶蜥蜴、冰原狼） | 4方向 | left/right 可镜像复用 |
| 漂浮型（怨魂） | 4方向 | Idle帧可复用，仅朝向不同 |
| 机械体（机械残骸、石像鬼、机械构造体） | 4方向 | left/right 可镜像复用 |
| 元素体（熔岩元素） | 4方向 | 全方向须独立绘制（内部纹理不对称） |
| BOSS | 4方向 (down/up/left/right) | 不可镜像（体型不对称） |

│                  │ 被晶化能量侵蚀后失去心智                    │
│─────────────────────────────────────────────────────────────│
│  体型            │ 人形，2.5头身，驼背微曲，双臂前伸           │

#### 3.1.2 逐帧动画定义

##### Idle (待机) — 6帧 @8FPS

```
帧1（0ms）:   站立中立位，驼背，双臂微垂，身体重心略前倾
帧2（125ms）: 身体微抬(+1px)，胸腔吸气扩张(+1px)，手臂微颤
帧3（250ms）: 呼吸顶点：身体抬至最高(+2px)，头部微仰(+1px)
帧4（375ms）: 身体回落(-1px)，手臂下垂摆动(-1px)
帧5（500ms）: 呼气末：身体微沉(-1px)，头部低垂(-1px)，肩膀耸动
帧6（625ms）: 复位至中立位，身体微晃(+0.5px)，循环回帧1
───────────────────────────────────────────────────────────────
总时长: 750ms/循环  |  位移: ±2px (Y轴呼吸浮动)
特效: 身体边缘蓝紫光晕(#4A3A8C, α=15%)微呼吸(亮度随呼吸±10%)
```

##### Walk (行走) — 6帧 @10FPS

```
帧1（0ms）:   右脚前迈着地，左脚在后，身体前倾5°，双臂前伸
帧2（100ms）: 重心前移，左脚离地，右脚承重，身体起伏-1px
帧3（200ms）: 左脚前迈至最前，悬空，右脚蹬地，身体上升+1px
帧4（300ms）: 左脚着地，身体下落-1px，手臂因惯性前甩+2px
帧5（400ms）: 重心转移至左脚，右脚跟离地，身体微晃
帧6（500ms）: 右脚前迈(同帧1姿态)，循环
───────────────────────────────────────────────────────────────
总时长: 600ms/循环  |  位移: 水平 3px/帧 (18px/循环)
特效: 脚下拖出微弱暗色残影(#1A1A2E, α=10%, 持续200ms)
```

##### Attack (攻击) — 6帧 @12FPS

```
帧1（0ms）:   【蓄力】身体后仰-2px，右手高举过头，五指张开
帧2（83ms）:  蓄力最大：身体后仰至极限(-3px)，手举最高(+4px)
帧3（167ms）: 【挥砍】身体前冲+3px，右手猛力下挥，指甲变长(+3px)
              指甲颜色：深灰#3A3A3A 渐变至 暗红#8B0020 尖端
帧4（250ms）: 攻击命中点：手臂伸展至最前(+5px)，身体前倾至极限
              命中特效：3道抓痕轨迹(#8B0020, α=60%, 持续200ms)
帧5（333ms）: 【收招】手臂回弹-2px，身体开始后撤
帧6（417ms）: 复位至中立位，手臂恢复自然前伸
───────────────────────────────────────────────────────────────
总时长: 500ms  |  攻击范围: 前方 12px 扇形(约60°)
伤害帧: 帧4  |  前摇: 250ms  |  后摇: 167ms
```

##### Hurt (受击) — 3帧 @12FPS

```
帧1（0ms）:   身体闪白(#FFFFFF, α=50% overlay)，后仰-3px
帧2（83ms）:  闪白消退(α=20%)，身体继续后仰(-4px)，手臂后甩
帧3（167ms）: 闪白消失，身体回弹+1px，准备恢复Idle
───────────────────────────────────────────────────────────────
总时长: 250ms  |  位移: -4px (击退方向)
```

##### Death (死亡) — 6帧 @8FPS

```
帧1（0ms）:   身体僵直，手臂垂下，头部后仰
帧2（125ms）: 身体透明化开始(α=85%→70%)，边缘蓝紫辉光增强(+30%)
帧3（250ms）: 身体前倾崩解：从脚部开始粒子化(暗红粒子#8B0020)
帧4（375ms）: 身体崩解至腰部，上半身透明化(α=50%)，粒子向上扩散
帧5（500ms）: 仅剩头部和手臂轮廓，大量蓝紫升腾粒子(#4A3A8C)
帧6（625ms）: 完全崩解：身体消失，遗留15-20个粒子消散(持续500ms)
───────────────────────────────────────────────────────────────
总时长: 750ms  |  崩解粒子: 暗红#8B0020(60%) + 蓝紫#4A3A8C(40%)
粒子大小: 2-4px菱形  |  生命周期: 0.5-1.0秒  |  扩散方向: 向上+向外
```

##### Aggro (警觉) — 4帧 @10FPS

```
帧1（0ms）:   身体猛震+1px，头部快速转向玩家方向
帧2（100ms）: 身体前倾，双臂上抬至攻击预备位(+2px)，眼睛闪红(#C41E3A)
帧3（200ms）: 发出无声嚎叫：嘴部张开(+2px)，身体前倾+2px
帧4（300ms）: 进入战斗姿态：锁定目标，手臂前伸，循环进入Walk或Attack
───────────────────────────────────────────────────────────────
总时长: 400ms  |  触发表情符号: 红色感叹号(!)图标(头顶12px处, 持续500ms)
```
