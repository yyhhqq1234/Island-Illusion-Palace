**《幻宫：时空回响》结局判断与触发方案（精确参数版）**

**一、结局决定函数**

```
结局 = f(灵魂核心碎片数, 记忆碎片数, 最终负担值, 最终战选择)
```

**参数权重分配**

  ---------------------------------------------------------------
  参数             取值范围          影响权重      数据来源
  ---------------- ----------------- ------------- -------------
  **核心碎片**     0 - 3             40%           击败关键BOSS获取

  **记忆碎片**     0 - 7             30%           记忆守护者掉落

  **最终负担**     0 - 100           20%           BurdenSystem

  **最终选择**     QTE 成功 / 失败    10%           最终战交互
  ---------------------------------------------------------------

**二、四大结局精确判断矩阵**

**结局 A：完美重构（真结局）**

  -------------------------------------------------------
  条件项                   阈值要求
  ------------------------ ------------------------------
  核心碎片                  3（全部）

  记忆碎片                  ≥ 5

  最终负担                  < 30

  最终战 QTE                "灵魂共鸣仪式"成功

  难度要求                  疯狂模式（Hardcore）
  -------------------------------------------------------

> **剧情**：墨语成功重构完整莎娜灵魂。莎娜保留全部记忆，力量完全恢复。两人共同稳定幻宫，成为时空守护者。
> **奖励**：真结局成就；莎娜主题皮肤；解锁隐藏难度"永恒"；主菜单背景变为重构后的幻宫。

**结局 B：残缺回归（标准结局）**

**B1 变种（记忆碎片 = 3-4）**

  -------------------------------------------------------
  条件项                   阈值要求
  ------------------------ ------------------------------
  核心碎片                  = 3

  记忆碎片                  3 - 4

  最终负担                  ≤ 70
  -------------------------------------------------------

> **剧情**：莎娜复活但缺失情感与牺牲相关记忆，与墨语关系疏远，需重新建立羁绊。

**B2 变种（记忆碎片 = 5-6）**

  -------------------------------------------------------
  条件项                   阈值要求
  ------------------------ ------------------------------
  核心碎片                  = 3

  记忆碎片                  5 - 6

  最终负担                  30 - 70
  -------------------------------------------------------

> **剧情**：莎娜复活后忘记牺牲记忆，性格更乐观但对幻宫成因困惑。墨语独自承担真相。

**B3 变种（高负担净化）**

  -------------------------------------------------------
  条件项                   阈值要求
  ------------------------ ------------------------------
  核心碎片                  = 3

  最终负担                  > 70 但已完成净化仪式

  记忆碎片                  任意
  -------------------------------------------------------

> **剧情**：通过炼金净化强行降低负担完成复活，莎娜身体虚弱需长期恢复。

**结局 C：力量反噬（疯狂结局）**

  -------------------------------------------------------
  条件项                   阈值要求
  ------------------------ ------------------------------
  最终负担                  > 70 且未净化

  或 禁忌炼金               使用 ≥ 5 次"灵魂献祭"类配方

  或 核心碎片               < 3 但强行进行复活仪式
  -------------------------------------------------------

> **剧情**：墨语被死灵圣王之眼反噬，意识侵蚀，成为新的灾难源。幻宫崩坏，威胁其他宇宙容器。
> **特效**：通关后主菜单背景变为扭曲幻宫，BGM 变调，需要删除存档才能恢复。

**结局 D：牺牲救赎（理智结局）**

  -------------------------------------------------------
  条件项                   阈值要求
  ------------------------ ------------------------------
  最终战选择                主动选择"接受现实"

  记忆碎片                  ≥ 5

  幻宫稳定仪式              已完成（灵魂精华×25）
  -------------------------------------------------------

> **剧情**：墨语接受莎娜已死的事实，将碎片用于稳定幻宫，成为孤独的时空守护者。所有记忆片段在结局动画中闪回消散。

**三、结局判定流程**

```text
1. 检查是否触发"力量反噬"（优先级最高）
   → 负担 > 70 且未净化？→ 结局 C
   → 禁忌炼金 ≥ 5 次？→ 结局 C
   → 核心碎片 < 3 但强制复活？→ 结局 C

2. 检查最终战选择
   → 选择"接受现实"？→ 结局 D（需记忆碎片 ≥ 5 + 仪式完成）

3. 检查完美重构条件
   → 核心=3 且 记忆≥5 且 负担<30 且 QTE成功 且 疯狂难度？→ 结局 A

4. 其余全部 → 结局 B（按记忆碎片数细分 B1/B2/B3）
   → 记忆碎片 3-4 → B1
   → 记忆碎片 5-6 → B2
   → 高负担净化 → B3
```

**四、隐藏结局：时空旅者**

  -------------------------------------------------------
  条件项                   阈值要求
  ------------------------ ------------------------------
  多周目累计               至少 3 个周目收集全碎片（3核心+7记忆）

  炼金配方                 全部 14 个配方已解锁

  难度                     疯狂模式下通关

  负担挑战                 全程负担始终 < 20

  隐藏材料                 发现并合成"超时空道标"（隐藏配方）
  -------------------------------------------------------

> **超时空道标合成配方**：悖时薄片×3 + 时空碎片×9 + 灵魂精华×25（总价值 40×3 + 12×9 = 228，仅在实验室碎片区域可合成）
> **剧情**：墨语发现穿越回 ISLAND1 时间线的可能性，开放式结尾暗示续作。

**五、最终战：Scarlet Soul Shana (S-SN) 阶段与结局联动**

  -----------------------------------------------------------------------------------------------------------------------
  阶段         生命范围      攻击模式                         结局联动
  ------------ ------------- -------------------------------- ----------------------------------------------------------
  **阶段一**   100%-70%      灵魂弹幕 + 镰刀挥砍              无特殊联动（通用阶段）

  **阶段二**   70%-40%       7种记忆类型轮换攻击              若玩家携带对应记忆碎片，可触发"记忆共鸣反击"（额外伤害窗口）

  **阶段三**   40%-10%       全屏 AOE + 时空裂隙 + 地图收缩    若负担 < 30，地图收缩速度减半

  **阶段四**   10%-0%        根据碎片数量动态变化：            此阶段末尾触发 QTE 和最终选择：
                            • 碎片 < 3：狂暴无弱点             选择"执行仪式"→ 进入结局判定（A/B/C）
                            • 碎片 3-6：弱点可见               选择"接受现实"→ 结局 D
                            • 碎片 = 7：莎娜协助玩家战斗       
  -----------------------------------------------------------------------------------------------------------------------

**阶段四的 QTE 判定**

-   **灵魂共鸣仪式 QTE**：连续 3 轮按键序列（每轮 5 个按键），允许 2 次失误。
    -   全部成功 → 仪式完成
    -   失败 1 轮 → 仪式部分完成（影响结局 B 的质量）
    -   失败 2 轮 → 仪式失败（强制触发结局 C）
