**二、 敌人行为模式详解**

**2.1 基础行为模式库**

**追击型**

-   **移动模式**：直线接近玩家，受障碍物影响转向

-   **攻击模式**：近战接触伤害，简单攻击动画

-   **变种**：快速追击（速度+30%）、跳跃追击（可越过障碍）

-   **代表敌人**：腐化村民、结晶蜥蜴

**狂暴型**

-   **触发条件**：生命值低于30%或受特定攻击

-   **行为变化**：攻击速度+50%，移动速度+20%，防御-30%

-   **视觉提示**：身体变红，攻击特效增强

-   **代表敌人**：晶化狼、腐化狂战士

**伏击型**

-   **隐藏状态**：与环境融合，玩家靠近前静止

-   **突袭动作**：短距离冲刺+高伤攻击，冷却8-12秒

-   **警戒范围**：半径3-5米

-   **代表敌人**：沼泽潜伏者、暗影潜伏者

**召唤型**

-   **召唤物**：低级敌人（1-3个）

-   **召唤间隔**：15-25秒，需读条（可打断）

-   **召唤上限**：同时存在2组召唤物

-   **代表敌人**：灵魂祭司、机械构造体

**远程型**

-   **攻击距离**：10-15米

-   **弹道类型**：直线、抛物线、追踪

-   **移动策略**：保持距离，被接近时后撤

-   **代表敌人**：晶能弓手、机械炮台

**战术型**

-   **技能组合**：2-3个技能循环使用

-   **团队协作**：多敌人时形成配合（坦+输出+控制）

-   **弱点保护**：会优先保护弱点暴露的队友

-   **代表敌人**：机械战术小队、灵魂守卫

**3.2 敌人弱点设计规则**

-   每个敌人有1-3个**弱点属性**（伤害×1.8）

**5.1 BOSS类型与阶段**

  ---------------- ------------- ------------------------ ------------------------

**5.2 BOSS战通用设计原则**

1.  **可学习性**：每个攻击都有明显前摇和提示

2.  **模式化**：攻击模式循环，便于玩家熟悉

3.  **弱点窗口**：每次攻击后都有短暂脆弱期

4.  **阶段清晰**：阶段转换有明确视觉和音频提示

5.  **检查点**：多阶段BOSS在阶段转换时设置检查点

**5.3 最终BOSS：Scarlet Soul Shana (S-SN)设计**

**阶段一：灵魂碎片（100%-70%生命）**

-   **攻击模式**：远程灵魂弹幕，近战镰刀挥砍

-   **弱点**：圣光属性

-   **特殊机制**：召唤莎娜记忆幻影（需优先击杀）

**阶段二：记忆回溯（70%-40%生命）**

-   **攻击模式**：记忆碎片攻击（7种记忆类型轮换）

-   **弱点**：根据当前记忆类型变化

-   **特殊机制**：玩家需使用对应记忆碎片共鸣进行反击

**阶段三：时空融合（40%-10%生命）**

-   **攻击模式**：全屏AOE，时空裂隙攻击

-   **弱点**：灵魂属性

-   **特殊机制**：地图不断收缩，玩家需在限定区域内战斗

**阶段四：最终抉择（10%-0%生命，根据碎片数量变化）**

-   **攻击模式**：根据玩家收集的记忆碎片数量决定

    -   碎片\<3：狂暴攻击，无弱点

    -   碎片3-6：攻击减弱，弱点明显

    -   碎片7：莎娜意识恢复，协助玩家战斗

-   **特殊机制**：玩家可选择击杀或救赎，触发不同结局

```pseudocode
FUNCTION UpdateBehavior():
    // 1. 狂暴检测（血量<30%）
    IF currentHealth / maxHealth <= berserkHealthThreshold AND NOT isBerserk:
        EnterBerserk()
    
    // 2. 逃跑检测（低血量+远程/战术型）
    IF currentHealth / maxHealth <= lowHealthThreshold 
       AND (behaviorMode == Ranged OR behaviorMode == Tactical):
        isFleeing = true
        FleeBehavior()
        RETURN
    
    // 3. 按基础行为模式执行
    SWITCH behaviorMode:
        CASE Pursuit:   ChaseAndAttackPlayer()
        CASE Berserk:   AggressiveCharge()
        CASE Ambush:    WaitAndAmbush()
        CASE Summoner:   SummonMinions()
        CASE Ranged:    KeepDistanceAndShoot()
        CASE Tactical:  ExecuteTacticalAI()

FUNCTION EnterBerserk():
    isBerserk = true
    attackRate *= 1.5      // 攻击速度+50%
    moveSpeed *= 1.2        // 移动速度+20%
    // defense *= 0.7       // 防御-30%
    // 视觉反馈：身体变红
    spriteRenderer.color = Color.red
```

**8.3 Boss多阶段切换（BossAI）**

```pseudocode
// 文件：Assets/Scripts/Enemy/BossAI.cs
// 实现路径：已有BossAI框架，需按设计文档补全阶段逻辑

FUNCTION UpdatePhase():
    healthPercent = currentHealth / maxHealth
    
    currentPhase = 
        healthPercent > 0.70 ? Phase1 :
        healthPercent > 0.40 ? Phase2 :
        healthPercent > 0.10 ? Phase3 :
        Phase4
    
    IF currentPhase != lastPhase:
        OnPhaseTransition(currentPhase)
        lastPhase = currentPhase

FUNCTION OnPhaseTransition(newPhase: int):
    // 阶段转换：清理旧技能、添加新技能、播放特效
    Debug.Log($"BOSS进入第{newPhase}阶段！")
    ShowPhaseChangeEffect()
    
    SWITCH newPhase:
        CASE 2:
            AddAttackPattern("记忆碎片攻击")
            SwitchWeakness()  // 弱点变化
        CASE 3:
            AddAttackPattern("全屏AOE")
            AddAttackPattern("时空裂隙")
            StartMapShrink()  // 地图缩小
        CASE 4:
            IF collectedFragments >= 7:
                EnableAllyMode()  // 莎娜协助玩家
            ELSE:
                EnableTrueForm()  // 狂暴最终形态

// ★ S-SN最终战阶段四特殊逻辑
FUNCTION Phase4Behavior():
    IF collectedCoreFragments < 3:
        // 无弱点狂暴模式
        weaknesses.Clear()
        damage *= 1.5
    ELSE IF collectedFragments == 7:
        // 莎娜意识觉醒，协助玩家
        SPAWN AllyShana at player side
        bossDamage *= 0.7  // BOSS伤害降低
```

```pseudocode
public class DynamicDifficulty : MonoBehaviour:
    float winStreak = 0
    float loseStreak = 0
    float areaClearTime = 0
    
    FUNCTION RecordBattleResult(won: bool):
        IF won:
            winStreak++
            loseStreak = 0
            IF winStreak >= 5: AdjustDifficulty(1.10, "连胜加成")
            IF winStreak >= 10: AdjustDifficulty(1.20, "十连胜！")
        ELSE:
            loseStreak++
            winStreak = 0
            IF loseStreak >= 3: AdjustDifficulty(0.90, "连败保护")
    
    FUNCTION RecordAreaClear(elapsedTime: float):
        IF elapsedTime < 180:  // 3分钟内清图
            nextAreaEliteDensity *= 1.4
    
    FUNCTION AdjustDifficulty(multiplier: float, reason: string):
        // 调整下一区域所有敌人属性
        difficultyMultiplier *= multiplier
        Debug.Log($"动态难度调整：{reason}，倍率 {difficultyMultiplier}")
        // 在生成新区域敌人时应用
        FOR EACH enemy IN nextAreaEnemies:
            enemy.damage *= multiplier
            enemy.maxHealth *= multiplier
```

```pseudocode
FUNCTION RollLoot(enemy: EnemyAI) -> List<LootEntry>:
    loot = new List<LootEntry>()
    
    // 1. 100%掉落灵魂
    loot.Add(LootEntry("灵魂", enemy.soulDropAmount))
    
    // 2. 灵魂之核掉落
    coreQuality = enemy.enemyQuality  // Common/Elite/Boss
    IF coreQuality != null:
        loot.Add(LootEntry("灵魂之核", coreQuality))
    
    // 3. 灵魂精华（仅精英/BOSS）
    IF enemy.isElite:
        IF Random.value <= 1.0:  // 100%
            loot.Add(LootEntry("灵魂精华", enemy.essenceDropAmount))
    
    // 4. 记忆碎片（仅记忆守护者）
    IF enemy.enemyType == MemoryGuardian:
        IF Random.value <= 0.20:  // 20%
            loot.Add(LootEntry("记忆碎片", memoryType))
    
    // 5. 材料掉落（按概率表）
    SWITCH enemy.enemyType:
        CASE CorruptedVillager:
            IF Random.value <= 0.30: loot.Add(CorruptedTissue, 1)
            IF Random.value <= 0.15: loot.Add(BoneFragments, 1)
        CASE CrystalLizard:
            IF Random.value <= 0.10: loot.Add(CrystalizedCore, 1)
        // ... 其余敌人
    
    RETURN loot
```
