# 02 — 人物设定

> ⚠️ **适用作品：《ISLAND：Blossom》**
>
> 本文档仅收录在《ISLAND：Blossom》中登场的主要人物，以其在 Blossom 故事时间段（2316-2333）内的状态为准。

---

## 一、主角

### 墨语（MY）

**身份**：S-C帝国三皇子，死灵圣法神 D·chil 契约继承者

**背景**：
- 2316年"紫星陨落之夜"诞生时左手即佩戴"死灵圣王之眼"戒指，与最古老神明 D·chil 的契约就此缔结
- 自幼因死灵之力被视为不祥，遭受宫廷排斥
- 两岁时母亲墨泉白因契约反噬去世
- 六岁时因触碰禁书险致灵魂溃散，被父亲暮晨放逐至北部边境
- 在传奇工匠"创之人"门下修炼八载，学习锻造、炼金、剑术
- 理解死灵之力的本质——非为毁灭，而是"引渡"与"守望"
- 亲手锻造出具有灵魂共鸣的佩剑"叙写者"

**外貌特征**：

| 属性 | 描述 |
|------|------|
| 年龄 | 16岁（故事主要阶段） |
| 身高 | 178cm |
| 体型 | 修长挺拔，肩宽腰窄，肌肉线条流畅清晰 |
| 发色 | 深黑色短发，质地偏硬，略显凌乱 |
| 瞳色 | **异色瞳**：右眼"星夜紫"（灵魂能力与契约体现），左眼深褐色 |
| 标志特征 | 左手佩戴"死灵圣王之眼"戒指（不化为印记），双手有锻造硬茧 |
| 气质 | 沉静坚韧，超越年龄的成熟中萦绕孤寂感 |
| 服饰 | 深色系简练工装、修身长裤与短靴；正式场合穿皇室暗纹简约礼服 |

## 二、核心女性角色

### 莎娜·绯红（SN）

**身份**：绯红家族第七代家主漓茉之女，墨语的未婚妻与守望者

**背景**：
- 比墨语年长一岁
- 三岁觉醒通灵之力，能通过灵魂光芒颜色判断强弱（白光 < 蓝光 < 紫光 < 红光 < 金光 < 黑色无法判断）
- 精通剑术、枪械、药剂与灵魂领域知识，家族百年不遇的天才
- 自幼接受严苛的守望者训练，心智沉静早熟，执行力极强
- 通过暮宇远程指导墨语掌控死灵之力
- 代表古老"守望亡者之契"的现世履行，与墨语的联系超越婚约，是灵魂层面的守护同盟

**外貌特征**：

| 属性 | 描述 |
|------|------|
| 年龄 | 17岁 |
| 身高 | 172cm |
| 体型 | 高挑纤秾合度，柔美而富有爆发力的曲线 |
| 发色 | 及腰绯红色长直发，光泽如缎 |
| 瞳色 | "绯红之瞳"，如红宝石或燃烧的彼岸花 |
| 标志特征 | 常佩戴彼岸花造型饰品 |
| 气质 | 神秘锐利，沉静早熟 |
| 服饰 | 训练时深红/黑色劲装；正式场合绯红与玄黑礼服 |

### 罗法·西亚 / 提娜·卡斯特（Lofa Seya）

**身份**：卡斯特家族嫡女，流亡者

**背景**：
- 原名提娜·卡斯特，祖父曾与 S-SN 约定，以背负诅咒为代价换取灵魂石与灵魂奥秘
- 祖父创立"Soul"研究灵魂，后遭圣庭打击解散
- 提娜降生后诅咒应验，灵魂中有 S-SN 的一部分，无法接受圣庭洗礼
- 称"灾厄"，流放后改名 Lofa·Seya
- 携带灵魂石项链和秘银匕首，在 S-C 与 N-U 交界处邂逅墨语

**外貌特征**：

| 属性 | 描述 |
|------|------|
| 年龄 | 15-16岁（初次登场） |
| 身高 | 170cm |
| 发色 | 雪白色长发（诅咒所致） |
| 瞳色 | 赤红色瞳孔（暗红如凝固血） |
| 标志特征 | 颈间佩戴灵魂石灰色项链，白发赤瞳的组合充满非人感与悲剧美 |
| 服饰 | 朴素耐磨的旅行者装束（灰、褐、深蓝），旧而整洁 |

### 伊卡洛斯 / "纤丝"

**身份**：WX 养女，王之刃成员，墨语的贴身护卫

**背景**：
- 原名伊卡洛斯·阿尔伯特。对墨语而言，她是"守护天使"——自幼看护、陪伴、以命相护的存在。异端之魂内部代号"织梦者"，王之刃代号"纤丝"，墨语昵称"伊诺"。
- 比墨语大4岁，从小与墨语一起生活、玩耍
- 8岁开始照顾墨语生活起居
- 墨语被放逐后跟随 WX 学习炼金术，展现极高天赋
- 后前往 N-U 进修，接触"Soul"关于灵魂的研究
- 得到 S-SN 启示后，创造性地将灵魂与炼金术结合，研制出炼金傀儡和傀儡之种
- 逃避圣庭追捕回到 S-C，在 WX 引荐下加入"王之刃"，代号"纤丝"
- 墨语昵称她为"伊诺"

**外貌特征**：

| 属性 | 描述 |
|------|------|
| 年龄 | 20岁 |
| 身高 | 168cm |
| 体型 | 匀称苗条，动作轻盈精准 |
| 发色 | 柔亮浅金色长发，常编精致发髻 |
| 瞳色 | 温暖琥珀色，目光清澈灵动 |
| 标志特征 | 面容柔和，常带鼓励性微笑；指尖有炼金痕迹 |
| 气质 | 亲切信赖，在护卫的专业与"姐姐"般的温柔间切换 |
| 服饰 | 任务时穿"王之刃"便装或侍从制服；私人时间着舒适简裙或学者袍 |

## 三、皇室暮家

### 暮晨

**身份**：南方国度君主，墨语、暮宇、暮雪之父

| 属性 | 描述 |
|------|------|
| 年龄 | 50余岁 |
| 身高 | 185cm |
| 发色 | 灰白相间短发，两鬓斑白 |
| 瞳色 | 深褐色（如墨语左眼），常显疲惫浑浊 |
| 标志特征 | 面容深刻，眉锁忧虑，法令纹深 |
| 气质 | 帝王威严与沉重疲惫交织，在父爱与国事间饱受煎熬 |

### 墨泉白（已故）

**身份**：暮晨之妻，神使一族长女，墨语与暮宇之母

**背景**：
- 拥有窥见神明意志的血脉能力
- 分娩墨语时目睹死灵圣法神降临并订立契约
- 产后因契约反噬，生命力持续流逝，墨语两岁时病逝
- 离世是墨语早期被孤立的重要转折点

| 属性 | 描述 |
|------|------|
| 发色 | 月光般的银色长发 |
| 瞳色 | 温柔"星夜紫"（比墨语更浅、更澄澈） |
| 标志特征 | 肤色白皙近透明，仿佛笼着柔光 |
| 气质 | 空灵圣洁，笑容温暖哀愁 |

### 暮宇

**身份**：南方国度大皇子，生灵圣法神 L·ries 契约继承者

**背景**：
- 比墨语年长十三岁
- 额头佩戴"生灵圣王之冠"
- 早慧睿智，极具政治远见与责任感
- 自幼坚定维护被孤立的弟弟墨语
- 在朝堂上与保守派周旋，力推改革与边境防务
- 是墨语与莎娜之间通信的桥梁，为弟弟提供精神指引与庇护

| 属性 | 描述 |
|------|------|
| 年龄 | 29岁 |
| 身高 | 188cm |
| 体型 | 高大挺拔，肩宽腰窄，体态健美雍容 |
| 发色 | 深棕色短发，一丝不苟 |
| 瞳色 | 清澈碧蓝色，目光温和睿智 |
| 标志特征 | 额头佩戴银白头环"生灵圣王之冠" |
| 气质 | 成熟可靠，兼具领袖风范与兄长温暖 |

### 暮雪

**身份**：南方国度公主（庶出），墨语的姐姐

**背景**：
- 比墨语年长两岁
- 墨语被软禁时期成为他最持续的陪伴者与保护者
- "书库事件"中为守护弟弟强行激活冰霜元素天赋
- 成年后协助暮宇，与绯红家族建立了秘密联络渠道
- 是墨语与皇室核心之间的情感纽带和信息桥梁

| 属性 | 描述 |
|------|------|
| 年龄 | 18岁 |
| 身高 | 170cm |
| 体型 | 高挑纤细，线条优美 |
| 发色 | 及肩银灰色直发（受天赋影响） |
| 瞳色 | 浅灰色，如冬日晨雾，清澈冷静中蕴藏温柔 |
| 气质 | 外柔内刚，温柔外表下是保护家人的坚韧 |

## 四、导师与盟友

### 创之人

**身份**："黑铁之心"创始人，墨语的主要导师

**背景**：
- 隐居北部边境的传奇工匠
- 教学注重实践与心性打磨，传授"倾听材料灵魂之声"的锻造哲学
- 引导墨语理解死灵之力乃"引渡与守望"
- 真实身份是维护世界稳定的守护者（Debugger的协助人）
- 知晓多元可能性的存在，负责引导"关键变量"平稳融入世界规则

| 属性 | 描述 |
|------|------|
| 年龄 | 外50-60岁 |
| 身高 | 约175cm |
| 体型 | 敦实强健如岩石，肩宽臂粗，手掌布满老茧伤疤 |
| 发色 | 灰白色短发与络腮胡，修剪整齐 |
| 瞳色 | 深邃灰蓝色，如古井或深海，洞悉本质 |
| 气质 | 质朴神秘，平常似沉默匠人，谈及"平衡"时威严尽显 |

### 沃希（WX / WonderX / WDR-010）

**身份**：皇家炼金师，伊卡洛斯的养父

**背景**：
- 劳伦斯家族人造神计划 Wonder Project 实验产物
- 心脏为"贤者之石"（sampo + 金属 + 晶 + 炼金法阵），拥有强大炼金能力
- 化名沃希潜逃进入 S-C，凭借炼金术入驻 S-C 皇室
- 见证了墨语和暮宇的出生，被墨语视为启蒙导师
- 后被暮雪派去支援墨语的 ISLAND 基建

| 属性 | 描述 |
|------|------|
| 年龄 | 外表30-35岁 |
| 身高 | 180cm |
| 体型 | 瘦高单薄，姿态挺直 |
| 发色 | 深棕色短发，两鬓夹银丝 |
| 瞳色 | 异色瞳：左眼深褐色；右眼为精密机械结构，瞳孔嵌淡金炼金法阵 |
| 标志特征 | 胸腔正中镶嵌"贤者之石"；身体约三成由炼金金属重构 |
| 气质 | 智慧神秘，略带玩世不恭 |
| 服饰 | 深色学者风外套、马甲，常佩单边眼镜掩饰右眼 |

### 漓茉·绯红

**身份**：绯红家族第七代家主，莎娜之母

**背景**：
- 墨语诞生当夜现身皇宫，出示"守望亡者之契"，确立家族守护职责
- 主动提出并缔结墨语与莎娜的婚约
- 作风果决，代表古老隐世家族的深厚底蕴与绝对承诺

| 属性 | 描述 |
|------|------|
| 年龄 | 外表35-40岁 |
| 身高 | 175cm |
| 发色 | 深红色长发，挽成华贵发髻 |
| 瞳色 | 暗红色瞳孔，比莎娜更沉郁深邃 |
| 标志特征 | 佩戴精致彼岸花金簪 |
| 气质 | 美艳绝伦，强大威严，眼神锐利如刀 |
