**《幻宫：时空回响》契约伙伴清单（详细终版）**

**总计：6个契约伙伴，均为永久型特殊召唤物，通过完成特定条件解锁**

**设计原则**

-   **获取方式**：每个契约伙伴需完成独立的支线任务线或满足特定条件解锁，不可直接通过战斗掉落获得。所有获取条件已兼容当前12种地图类型（7自然+3人文+2特殊+1最终）。

-   **永久性**：解锁后永久加入灵魂库，可跨周目保留（但需在新周目重新完成契约成本）。

-   **出战限制**：契约伙伴占用 1 个普通召唤位，与普通/精英/BOSS 召唤物共享 3 个出战位。

-   **独特机制**：每个契约伙伴拥有独立的 4 技能组（主动/被动混合），不同于普通召唤物的单一能力。

-   **数值联动**：契约伙伴与负担系统、记忆碎片系统、召唤系统存在数值联动规则（详见第七节）。

**五、契约伙伴升级**

-   **等级上限**：契约伙伴上限为 10 级（不同于普通白色5级/蓝色10级/金色15级）。

-   **升级消耗**：每级消耗灵魂精华×2 + 灵魂×（等级×20）。

-   **属性成长**：每级 +8%基础属性。

-   **突破**：5级时需突破（消耗灵魂精华×5），解锁第 4 个技能。

-   **跨周目**：契约伙伴等级跨周目不重置，但每局开始需重新满足契约成本才能出击。

**六、设计意图**

-   **支线驱动**：6个契约伙伴对应6条独立支线任务，提供探索动力和区域重玩价值。

-   **属性门槛**：契约成本设定属性/负担/精华门槛，使契约伙伴成为中后期强力单位。

-   **Build关联**：不同契约伙伴适配不同流派（伊格尼斯→物理、汐→召唤、苍穹之令→魔法）。

-   **跨周目延续**：等级保留递减重刷感，但出击条件限制避免开局碾压。

**七、契约伙伴与核心系统的数值联动规则**

契约伙伴作为特殊召唤物，与负担系统、记忆碎片系统、召唤系统存在以下数值联动规则，确保跨系统数值平衡。

**7.1 与负担系统的联动**

| 联动项 | 规则 | 数值 |
|--------|------|------|
| 出击负担消耗 | 每次出击契约伙伴时，玩家负担值 +5 | +5负担/次出击 |
| 高负担属性衰减 | 负担值 > 70 时，契约伙伴全属性 -15% | -15%全属性 |
| 负担临界惩罚 | 负担值 > 90 时，契约伙伴技能冷却时间 +50% | 冷却+50% |
| 负担清除恢复 | 在营火处清除负担后，契约伙伴属性恢复至正常 | 即时恢复 |

> 设计意图：负担系统作为全局风险管控机制，契约伙伴的高爆发/高辅助能力需要通过负担成本进行制衡。高负担时契约伙伴战斗力下降，鼓励玩家在出戰前规划负担管理。

**7.2 与记忆碎片系统的联动**

| 联动项 | 规则 | 数值 |
|--------|------|------|
| 记忆碎片共鸣增强 | 携带特定记忆碎片时，对应契约伙伴获得额外属性加成 | 见表7.2.1 |
| 碎片消耗触发 | 部分契约伙伴（罗法西亚）出击需消耗"牺牲"记忆碎片已激活 | 前置条件 |
| 碎片数量加成 | 每额外携带1个记忆碎片，契约伙伴技能效果 +3% | +3%/碎片 |

| 记忆碎片 | 共鸣契约伙伴 | 共鸣效果 |
|---------|------------|---------|
| 童年记忆 | 伊莉丝 | 治疗量+20% |
| 战斗记忆 | 伊格尼斯 | 火焰伤害+15% |
| 情感记忆 | 汐 | 召唤物持续时间+25% |
| 知识记忆 | 苍穹之令 | 雷电伤害+15% |

> 设计意图：记忆碎片与契约伙伴的共鸣配对鼓励玩家在Build时进行策略性搭配——选择携带哪个记忆碎片不仅影响探索/炼金，还影响契约伙伴的战斗力。单次最多携带3个记忆碎片（IIPConstants.MaxMemoryFragmentSlots），玩家需取舍。

**7.3 与召唤系统的联动**

| 联动项 | 规则 | 数值 |
|--------|------|------|
| 召唤位占用 | 契约伙伴占用1个普通召唤位（与普通/精英/BOSS召唤物共享3个出战位） | 1/3出战位 |
| 全局冷却共享 | 契约伙伴出击后，触发全局召唤冷却（与普通召唤相同） | 2秒冷却 |
| 召唤物属性光环 | 部分契约伙伴（汐、罗法西亚）的被动技能影响全场召唤物属性 | 见表7.3.1 |
| 召唤物数量上限 | 当出战位已满时，契约伙伴无法出击（需先召回其他召唤物） | 3召唤物上限 |

| 契约伙伴 | 光环效果 | 影响范围 |
|---------|---------|---------|
| 汐 | 所有召唤物获得水上行走能力 | 全场召唤物 |
| 罗法西亚 | 所有召唤物灵魂伤害+20% | 全场召唤物 |
| 伊莉丝 | 全场友方单位移动速度+10%（含召唤物） | 全场友方单位 |

> 设计意图：契约伙伴的光环效果是"召唤物军团Build"的核心驱动力。多个契约伙伴的光环可叠加，但受出战位（3个）限制，玩家需在"契约伙伴光环叠加"与"精英/BOSS召唤物单体战力"之间做取舍。

**7.4 数值联动规则总结**

```
契约伙伴出战决策树：
┌── 检查负担值 ──── 负担 > 70? ── YES ── 契约伙伴属性-15%（警告：战力下降）
│                                      └── 负担 > 90? ── YES ── 冷却+50%（严重警告）
│
├── 检查记忆碎片 ── 携带共鸣碎片? ── YES ── 对应契约伙伴获得额外加成
│
├── 检查召唤位 ──── 出战位已满? ── YES ── 需先召回其他召唤物
│
└── 执行出击 ────── 负担+5, 触发全局冷却2秒
```

**预期表现清单（数值联动）**：
- [ ] 低负担（<40）时出击契约伙伴，属性不受影响，可正常发挥
- [ ] 高负担（>70）时，契约伙伴全属性衰减15%，UI显示"负担压制"debuff图标
- [ ] 临界负担（>90）时，契约伙伴技能冷却时间+50%，UI显示"负担临界"警告
- [ ] 携带"童年记忆"碎片时，伊莉丝治疗量提升20%，数字显示为绿色加成
- [ ] 携带"牺牲记忆"碎片时，罗法西亚的出击条件"牺牲记忆碎片已激活"自动满足
- [ ] 同时携带3个记忆碎片且出战汐时，汐获得3个碎片的通用加成（+9%技能效果）+ 情感记忆共鸣加成
- [ ] 3个出战位中已有2个精英召唤物时，仅可再出击1个契约伙伴
- [ ] 在营火清除负担后，契约伙伴属性立即恢复正常
