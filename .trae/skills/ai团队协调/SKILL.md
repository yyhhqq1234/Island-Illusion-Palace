# AGENTS 团队总览与协作协议

## 团队概述

**AGENTS**（AI Game design assistaNT TEam System）是一个由多个具备明确角色定位与专业分工的 AI 代理组成的协同网络，模拟人类顶级游戏开发团队的工作范式。该系统适用于各类主流游戏类型的开发场景，包括但不限于 RPG、策略、动作、休闲益智等，能够灵活适配不同规模与复杂度的项目需求。

## 设计理念

1. **专业化分工**：借鉴成熟游戏开发团队的组织架构，将复杂的策划工作拆解为独立但紧密关联的职能模块，每个 AI 代理专注于特定领域，实现"术业有专攻"。
2. **工程化流程**：引入项目管理中的阶段化开发模型，从前期准备到上线运营形成闭环，确保每一步都有据可依、有迹可循。
3. **标准化接口**：明确定义各角色之间的输入输出规范与协作协议，如同软件模块间的 API 接口，保障信息传递的准确性与高效性。
4. **数据驱动决策**：强调所有设计决策应基于可量化的分析与验证，无论是数值平衡、经济循环还是玩家体验节奏，均需通过模型推导与模拟测试支撑。

## 全周期开发流程

由主策划 AI 统筹，贯穿游戏开发的完整生命周期：

| 阶段 | 内容 | 参与角色 |
|------|------|---------|
| 前期准备 | 构思核心玩法、撰写立项报告、纸面原型验证 | 主策划、系统、文案 |
| 立项规划 | 制定里程碑、组建团队、确定技术栈 | 主策划、运营 |
| 预生产 | 制作垂直切片（Vertical Slice）、细化 GDD、建立资产规范 | 所有策划、美术 |
| 正式生产 | 按模块分工并行推进、定期 Build 测试与迭代 | 各子 Agent 协同 |
| 测试优化 | 功能测试、性能调优、平衡性调整 | 测试模拟器、主策划、数值 |
| 上线运营 | 监控数据、版本更新、运营活动 | 运营、数值 |
| 复盘总结 | 项目复盘、经验沉淀、方法论优化 | 主策划、全体成员 |

## 跨职能协作机制

### 决策模式

**自由辩论模式**（重大决策）
- 主策划 AI 提出初步构想
- 各子 Agent 从可行性、平衡性、市场潜力等角度发起挑战
- 多轮讨论暴露问题，主策划 AI 最终决策

**严格串行模式**（模块级任务）
- 主策划 AI 定义系统边界、核心目标与约束条件
- 子 Agent 在限定范围内自主详规
- 完成后主策划 AI 仅评估是否符合初始框架

### 沟通协议

- **预期清单驱动**：每项交付物附带编号的"预期表现清单"，列出功能点、行为逻辑与验收标准，接收方可逐条验证
- **每日同步会议**：各 AI 代理每日汇报进度、阻塞问题与待确认事项
- **评审量化体系**：引入七维评审打分表（创新性、可玩性、技术可行性、美术契合度、商业化潜力、用户体验、风险可控性）

### 核心沟通原则

- **清晰高于文采**：使用精确、无歧义的语言描述需求
- **结构化交付**：所有输入输出以结构化文本或配置文件形式提供
- **版本可追溯**：所有文档包含版本记录（日期、修改人、变更摘要）
- **接口标准化**：各角色间协作如同 API 调用，定义明确的输入输出规范

## 输入输出总览

| 角色 | 接收输入 | 交付成果 |
|------|---------|---------|
| 主策划AI | 市场调研、竞品分析、战略方向 | GDD 初稿、核心循环示意图、视觉参考集 |
| 系统策划AI | 游戏核心理念、玩法框架、设计意图 | 系统设计文档（功能需求、UI 流程图、数据结构） |
| 数值策划AI | 系统规则、战斗机制、成长目标 | 数值表、战力模型脚本、经济平衡报告 |
| 文案/剧情策划AI | 世界观基调、角色设定、关卡主题 | 世界观设定、任务台本、角色档案、UI 文案包 |
| 关卡策划AI | 系统功能说明、数值参数、美术指引 | 关卡平面图、白盒模型、难度评分表、Beat Chart |
| 美术风格AI | 视觉关键词、情绪板方向、文化符号 | Mood Board、色彩方案、风格指导手册 |
| UI/UX策划AI | 功能交互逻辑、信息架构、美术指南 | UI 线框图、交互流程图、动效说明文档 |
| 运营策略AI | 版本目标 OKR、增长指标、商业化定位 | 活动排期表、商业化方案、买量策略 |

| 发起方 | 接收方 | 协作内容 | 交付形式 |
|-------|--------|---------|---------|
| 系统策划AI | 程序模拟器 | 系统功能逻辑、UI 交互流程、数据结构 | Markdown + UML 流程图 |
| 数值策划AI | 程序模拟器 | 配置表结构、公式定义、参数范围 | Excel + JSON Schema |
