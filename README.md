# Island Illusion Palace (幻宫·时空回响)

一款基于 Unity 引擎开发的 2D 角色扮演游戏。

## 游戏简介

《幻宫·时空回响》是一款融合了探索、战斗、炼金和召唤元素的 2D 角色扮演游戏。玩家将扮演一位神秘的炼金术士，在一个充满谜题和危险的奇幻世界中探索、收集材料、炼制药剂、召唤灵魂，并揭开隐藏在幻宫中的秘密。

## 核心玩法

### 🌲 探索
- 在森林等多个场景中自由探索
- 收集材料和发现隐藏的秘密
- 与环境互动，解开谜题

### ⚔️ 战斗
- 实时战斗系统
- 召唤灵魂辅助战斗
- 弱点洞察机制

### 🧪 炼金
- 收集各种材料进行炼金
- 自由炼金：尝试不同材料组合发现新配方
- 配方炼金：使用已知配方制作消耗品

### 👻 召唤
- 收集灵魂碎片
- 召唤不同类型的灵魂协助战斗
- 提升召唤物能力

## 技术栈

- **引擎**: Unity 2022.x
- **语言**: C#
- **渲染**: 2D Sprite

## 项目结构

```
Assets/
├── Animations/          # 动画资源
├── ArtMaterials/        # 美术素材
├── I-IP markdown/       # 设计文档
├── Music/               # 音乐和音效
├── Prefab/              # 预制件
├── Scenes/              # 场景文件
└── Scripts/             # C# 脚本
    ├── Enemy/           # 敌人相关
    ├── Systems/         # 系统类（炼金、背包等）
    ├── UI/              # UI 组件
    └── Utils/           # 工具类
```

## 快速开始

1. 克隆项目到本地
2. 使用 Unity 2022.x 打开项目
3. 运行 `MainMenu.unity` 场景开始游戏

## 游戏控制

- **WASD / 方向键**: 移动角色
- **鼠标**: 控制武器朝向
- **左键**: 攻击
- **E**: 互动（打开炼金面板等）
- **ESC**: 打开背包/暂停菜单
